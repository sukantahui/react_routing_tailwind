ভূমিকা (Introduction)

এই অধ্যায়ে আমরা আলোচনা করবো ডেটা স্ট্রাকচার ও তার বিভিন্ন প্রকারভেদ সম্পর্কে। প্রতিদিনের জীবন থেকে লাইব্রেরি ও ফ্রিজ-র মতো উদাহরণগুলোর মাধ্যমে ডেটা স্ট্রাকচারের ধারণা আরও সুস্পষ্ট করা হচ্ছে।

ডেটা টাইপসমূহ (Data Types)

Integer (পূর্ণসংখ্যা)

যে কোনো পূর্ণসংখ্যা, যেমন: 10, -50, 1050

Floating Point (দশমিক সংখ্যা)

যেমন: 45.02, 5.8, -1.2

Character (অক্ষর)

অক্ষর বা বিশেষ চিহ্ন: A, Z, 0-9, @, #, $ ইত্যাদি

Logical (লজিক্যাল)

True/False, Yes/No, On/Off

String (স্ট্রিং)

অক্ষর ও সংখ্যা সমন্বয়ে গঠিত একক ডেটা: "Rajib", "A007", "Computer"

লিনিয়ার ডেটা স্ট্রাকচার (Linear Data Structure)

যেখানে উপাদানগুলো একটির পর একটি সাজানো থাকে — যেমন: Array, Stack, Queue, Linked List।

অ্যারে (Array)

একই ধরনের উপাদানগুলির ধারাবাহিক সংগ্রহ। Syntax: int A[10];

Length নির্ণয়: Length = UB - LB + 1.

উদাহরণ: 
A = [11, 12, 13, 14, 15]
Index: 0  1   2   3   4
Length = 4 - 0 + 1 = 5

স্ট্যাক (Stack)

LIFO (Last In First Out) — Push ও Pop অপারেশন।

উদাহরণ: 
Push(A) → [A]
Push(B) → [A, B]
Push(C) → [A, B, C]
Pop()   → C removed → [A, B]

কিউ (Queue)

FIFO (First In First Out) — Head ও Tail ধারণা।

উদাহরণ: 
Enqueue(1) → [1]
Enqueue(2) → [1,2]
Enqueue(3) → [1,2,3]
Dequeue() → 1 removed → [2,3]

লিংকড লিস্ট (Linked List)

নোড-চেইন: প্রতিটি নোডে Data ও Next Address থাকে; Dynamic memory allocation।

উদাহরণ (সিঙ্গল লিংকড লিস্ট): 
Start -> [Info: 10 | Next: addr1] -> [Info: 20 | Next: addr2] -> [Info: 30 | Next: NULL]

নন-লিনিয়ার ডেটা স্ট্রাকচার (Non-Linear)

অলোমেলো বা হায়ারার্কিক্যালভাবে সাজানো ডেটা — উদাহরণ: Tree, Graph।

Tree (ট্রি)

Root এবং Child নোডের সাথে Hierarchical structure.

Graph (গ্রাফ)

Vertex ও Edge দ্বারা গঠিত সংযোগী নেটওয়ার্ক।
