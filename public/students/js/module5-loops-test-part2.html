Q1 ¬∑ Output Prediction

What will be the output of the following code?

  console.log("Loop:", i);
}

  • b) Loop: 1, Loop: 2, Loop: 3

Q2 ¬∑ Loop Trace

Q3 ¬∑ Output

Q9 ¬∑ Infinite or Not?
while
Does this create an infinite loop? 
let i = 0;

while (i < 3) {
  console.log(i);
}
  • a) Yes, infinite loop.
  • b) No, runs 3 times.
  • c) No, runs 2 times.
  • d) No, runs 1 time.
Correct Answer: a)

There is no update to i inside the loop. i stays 0, condition is always true, so the loop never ends. This is a classic infinite loop bug.

Initialization runs once at the beginning of the for loop lifecycle. Then condition and update handle the rest of the iterations.
