Q1 ¬∑ Concept for loop basics

Which part of the for loop is executed only once? 

for (part1; part2; part3) {
  // body
}
  • A. part1 only
  • B. part2 only
  • C. part3 only
  • D. All three parts run on every iteration

Correct Answer: A

Explanation: In a for loop, initialization (part1) runs once at the start. Then each cycle: condition (part2) ‚Üí body ‚Üí update (part3).

Q2 ¬∑ Output for loop

What will this code print?

for (let i = 1; i <= 3; i++) {
  console.log(i);
}
  • A. 1 2
  • B. 1 2 3
  • C. 0 1 2
  • D. Infinite loop

Correct Answer: B

Explanation: i starts at 1, runs while i <= 3, then increments. So 1, 2, 3 are printed.

Q3 ¬∑ Output for loop

What is the output?

for (let i = 5; i > 0; i--) {
  console.log(i);
}
  • A. 5 4 3 2 1
  • B. 1 2 3 4 5
  • C. 4 3 2 1 0
  • D. No output

Correct Answer: A

Explanation: The loop starts at 5 and decrements until i becomes 0. Last valid value is 1, so prints 5 ‚Üí 1.

Q4 ¬∑ Concept while loop

Which loop is best when the number of iterations is not known in advance?

  • A. for loop
  • B. while loop
  • C. do‚Ä¶while loop only
  • D. No loop is suitable

Correct Answer: B

Explanation: while is typically used when you keep looping until a condition changes (e.g., user input), not a fixed count.

Q5 ¬∑ Output while loop

What will be printed?

while (i < 4) {
  console.log(i);
  i++;
}
  • A. 1 2
  • B. 1 2 3
  • C. 2 3 4
  • D. Infinite loop

Correct Answer: B

Explanation: Values printed: 1, 2, 3. When i becomes 4 the condition i < 4 is false so the loop stops.

Q6 ¬∑ Output do‚Ä¶while

What does this code output?

let n = 5;

do {
  console.log(n);
  n++;
} while (n < 5);
  • A. No output
  • B. 5
  • C. 5 6
  • D. Infinite loop

Correct Answer: B

Explanation: do‚Ä¶while runs the body once before checking the condition. So 5 is printed once, then n becomes 6 and the loop stops.

Q7 ¬∑ Concept syntax

Which of the following is a correct for loop syntax?

  • A. for i = 0 to 10 { }
  • B. for (let i = 0; i < 10; i++) { }
  • C. for (i < 10; i++) { }
  • D. for (let i = 0; i++; i < 10) { }

Correct Answer: B

Explanation: JavaScript requires the pattern for (initialization; condition; update) with semicolons.

Q9 ¬∑ Concept infinite loop

Which of the following will cause an infinite loop?

  • A. for (let i = 0; i < 5; i++)
  • B. while (false)
  • C. while (true) with no break
  • D. for (let i = 10; i > 0; i--)

Correct Answer: C

Explanation: while (true) never becomes false, so unless you manually break, it will never stop.

Q11 ¬∑ Logic even numbers

You want to print all even numbers from 2 to 20 (inclusive). Which loop header is correct?

  • B. for (let i = 0; i <= 20; i++)

Q13 ¬∑ Output while

  count--;

  • B. 2 1 0

Q14 ¬∑ Concept do‚Ä¶while

Which statement about do‚Ä¶while is true?

  • A. The condition is checked before the first run.
  • B. The body may never execute.

Explanation: do‚Ä¶while executes the body first, then checks the condition, guaranteeing at least one run.

Q24 ¬∑ Concept loop variable

Why is it safer to use let i = 0; instead of i = 0; inside a loop?

  • A. let is faster
  • B. let ensures block scope, preventing accidental reuse outside
  • C. i = 0; is invalid JavaScript
  • D. Both are identical everywhere

Correct Answer: B

Explanation: let declares a block-scoped variable. Using plain i = 0 can accidentally use or create a global variable.
