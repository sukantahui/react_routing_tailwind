Questions 51–75

51. Rewrite using a template literal:

"Hello " + user + ", your score is " + score

`Hello ${user}, your score is ${score}`
Explanation: Backticks and ${...} give cleaner, readable string building.

52. What is the output?

let user = "Rahul";
let city = "Kolkata";
console.log(`Welcome ${user} from ${city}`);
Answer: "Welcome Rahul from Kolkata"
Explanation: Both variables are interpolated inside the template literal.

53. Identify the bug:

let name = "Rita";
console.log('Hello, ${name}');
Answer: Wrong quotes
Explanation: Template placeholders ${name} only work with backticks, not single quotes.

54. Fix the above code.

let name = "Rita";
console.log(`Hello, ${name}`);

55. Output?

let x = 5;
let y = 2;
console.log(`Result: ${x ** y}`);
Answer: "Result: 25"
Explanation: 5 ** 2 = 25, then embedded in the template literal.

56. What does this print?

console.log("5" - "2");
Answer: 3
Explanation: - forces numeric conversion of both strings.

57. What does this print?

console.log("5" + "2");
Answer: "52"
Explanation: With strings, + concatenates.

58. Predict the output:

console.log("5" - 2, "5" + 2);
Answer: 3 "52"
Explanation: Left: numeric subtraction → 3. Right: string concatenation → "52".

59. Type of each?

typeof "5"        // ?
typeof 5          // ?
typeof true       // ?
typeof 123n       // ?
Answer: "string", "number", "boolean", "bigint"

60. Write code to log the data type of variable value.

console.log(typeof value);
Explanation: typeof operator returns a string representing the type.

61. What is printed?

let a = 10;
let b = 3;
console.log(a % b);
Answer: 1
Explanation: Remainder of 10 / 3 is 1.

62. True or False: console.log is only for debugging and not part of JS syntax rules.

Answer: True
Explanation: It’s an environment-provided function (like in browsers or Node), not a language keyword.

63. Complete the expression to convert input string s to number:

let n = ____________; (using Number)

let n = Number(s);

64. Complete to convert number n to string:

let s = ____________;

let s = String(n);

65. True or False: parseInt("10px") returns 10.

Answer: True
Explanation: parseInt reads numeric prefix until a non-digit character.

66. What will Number("10px") return?

Answer: NaN
Explanation: Number requires the entire string to be numeric.

67. Which operator is NOT logical?

A. &&   B. ||   C. !   D. ??

Answer: D (kind of trick: ?? is nullish coalescing, not a classic boolean logic operator)

68. Output?

console.log( true && false );
Answer: false

69. Output?

console.log( true || false );
Answer: true

70. Output?

console.log( !false );
Answer: true

71. Fill in: “If a is null or undefined, use 100 else use a”

let final = a ?? 100; – Is this correct?

Answer: Yes
Explanation: ?? uses the right-hand value only if left is null or undefined.

72. Simple output program: What is printed?

let celsius = 30;
let fahrenheit = (celsius * 9/5) + 32;
console.log(fahrenheit);
Answer: 86

73. Predict:

let radius = 3;
const PI = 3.1416;
console.log(PI * radius * radius);
Answer: Approximately 28.2744

74. True or False: A const variable’s object contents can still be changed.

Answer: True
Explanation: const prevents reassignment of the binding, not mutation of the object.

75. Short code: Create const GST = 18 and log final price for mrp = 100.

const GST = 18;
let mrp = 100;
let finalPrice = mrp + (mrp * GST / 100);
console.log(finalPrice); // 118
