Test (Part 2: Questions 26–50)

Premium Practice Test • Logical Operators • Comparison Logic • Template Literals • Boolean Evaluation
Q26. What is the output of:
console.log(true && true && false);
Answer: false

Explanation: AND (&&) returns true only if all values are true. Since the last value is false → entire expression is false.
Q27. Evaluate:
true || false || false
Answer: true

Explanation: OR (||) stops early (“short-circuiting”) when it finds the first true.
Q28. What is the result of:
!false && true
Answer: true

Explanation: !false becomes true → true && true → true.
Q29. Predict the output:
console.log(!(5 > 2));
Answer: false

Explanation: 5 > 2 is true → !true gives false.
Q30. Evaluate:
10 > 5 || 5 > 20
Answer: true

Explanation: First condition is true → OR stops immediately.
Q31. What does this return?
false && (10 / 0)
Answer: false

Explanation: AND short-circuits at the first false → second part never runs.
Q32. Evaluate:
true || (5 / 0)
Answer: true

Explanation: OR short-circuits when true appears → division never executes.
Q33. Output of:
console.log(!("hello"));
Answer: false

Explanation: "hello" is truthy → !truthy = false.
Q34. Is "" truthy or falsy?
Answer: falsy

Explanation: Empty string is one of JavaScript’s built-in falsy values.
Q35. Evaluate:
0 || 25
Answer: 25

Explanation: 0 is falsy → OR returns the next value.
Q36. What is:
"" && "JS"?
Answer: "" (empty string)

Explanation: AND returns the first falsy value.
Q37. Output of:
console.log("Hi" && 0);
Answer: 0

Explanation: AND returns first falsy → 0 is falsy.
Q38. Evaluate:
true && "JS"
Answer: "JS"

Explanation: AND returns last value if all are truthy.
Q39. Result of:
false || "Hello"
Answer: "Hello"

Explanation: OR returns first truthy → "Hello".
Q40. Evaluate:
!0
Answer: true

Explanation: 0 is falsy → !falsy = true.
Q41. Output:
console.log(!" ")
Answer: false

Explanation: A space (" ") is truthy → !truthy = false.
Q42. What is the result?
5 > 2 && 2 > 10
Answer: false

Explanation: Second comparison is false → entire AND is false.
Q43. Evaluate:
5 === "5" || 10 > 2
Answer: true

Explanation: First is false, but second is true → OR returns true.
Q44. Output:
console.log(!null);
Answer: true

Explanation: null is falsy → !falsy = true.
Q45. Which is truthy?
0, "", "0", null
Answer: "0"

Explanation: Non-empty strings are truthy even if they contain "0".
Q46. Evaluate:
(5 && 10) || 0
Answer: 10

Explanation: 5 && 10 → both truthy → returns 10. Then 10 || 0 → returns 10.
Q47. Result of:
(0 && 10) || 20
Answer: 20

Explanation: 0 && 10 → returns 0 (falsy). Then 0 || 20 → returns 20.
Q48. What does this give?
"JS" || false
Answer: "JS"

Explanation: OR returns first truthy → "JS".
Q49. Evaluate:
!undefined
Answer: true

Explanation: undefined is falsy → !falsy = true.
Q50. Output:
console.log(10 && "Hello" && true);
Answer: true

Explanation: All truthy → AND returns last value.
