Step 1 Â· What Are Loops?

1. Why Loops Are Needed

A loop lets you repeat a block of code multiple times without rewriting it. Loops are essential for tasks like printing tables, processing arrays, repeating calculations, or checking inputs.

1.1 Without Loops

console.log(1);
console.log(2);
console.log(3);
console.log(4);
console.log(5);

1.2 With a Loop

for (let i = 1; i <= 5; i++) {
  console.log(i);
}
Loops save time, reduce errors, and make code look professional.
