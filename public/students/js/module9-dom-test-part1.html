Module 9 — DOM Test Part 1
Selecting Elements

25 Questions on querySelector, getElementById, querySelectorAll, NodeList, element selection behaviour. Click each answer button to reveal explanation.

1. Which method selects an element by its ID?
A. query()
B. querySelector()
C. getElementById()
D. getById()
Answer: C
Explanation: getElementById("id") is the dedicated method to find elements by ID.
2. querySelector(".title") returns:
A. All elements with class="title"
B. First matching element
C. All <h1> elements
D. Only IDs
Answer: B
Explanation: querySelector returns only the first matching element.
3. querySelectorAll("p") returns:
A. Array
B. NodeList
C. Object
D. HTMLCollection
Answer: B
Explanation: querySelectorAll always returns a static NodeList.
4. Which selector is valid for querySelector?
A. "#box .item"
B. ".class1.class2"
C. "ul > li"
D. All of these
Answer: D
Explanation: querySelector supports full CSS selector syntax.
5. getElementById returns:
A. NodeList
B. HTMLCollection
C. Single element or null
D. Always undefined
Answer: C
Explanation: ID is unique; method returns the matching element or null.
6. Which is TRUE about NodeList?
A. It is always live
B. It is static when returned by querySelectorAll
C. It is an array
D. It has map()
Answer: B
Explanation: NodeLists from querySelectorAll are static, not live.
7. Which method supports CSS combinators?
A. getElementById
B. getElementsByTagName
C. querySelector
D. getElementsByClassName
Answer: C
Explanation: querySelector can use full CSS combinator syntax.
8. document.querySelector("#box") selects:
A. All elements with id="box"
B. First element with id="box"
C. All .box classes
D. Nothing
Answer: B
Explanation: IDs are unique, querySelector returns the first match.
9. Which returns a LIVE collection?
A. getElementsByClassName
B. querySelectorAll
C. NodeList
D. Array.from()
Answer: A
Explanation: HTMLCollection is live; querySelectorAll NodeLists are static.
10. Which is correct?
A. NodeList supports forEach()
B. NodeList supports map()
C. NodeList supports filter()
D. NodeList supports reduce()
Answer: A
Explanation: Modern NodeLists have forEach but not array methods like map.
11. querySelector("div p") selects:
A. All p inside any div
B. Only first p inside first div
C. All divs
D. First div
Answer: B
Explanation: querySelector returns only the first match of the full selector.
12. Which returns multiple results?
A. getElementById
B. querySelector
C. querySelectorAll
D. textContent
Answer: C
Explanation: querySelectorAll returns all matching nodes.
13. getElementsByTagName("p") returns:
A. NodeList
B. HTMLCollection
C. Array
D. Object
Answer: B
Explanation: getElementsByTagName returns a live HTMLCollection.
14. CSS selector "#id .item" means:
A. Select an ID
B. Select all .item inside #id
C. Select only first item inside #id
D. Select #id and .item
Answer: B
Explanation: The selector finds all .item elements that are descendants of #id.
15. document.body is:
A. NodeList
B. HTML element
C. Text node
D. Array
Answer: B
Explanation: document.body directly references the body element.
16. Which method allows advanced CSS selectors?
A. getElementsByTagName
B. querySelector
C. getElementById
D. innerText
Answer: B
17. NodeList is:
A. Array
B. Array-like object
C. String
D. HTMLElement
Answer: B
Explanation: NodeList is array-like but not a true array.
18. querySelector("#a, .b") selects:
A. Only #a
B. First match among #a or .b
C. All matches
D. Only classes
Answer: B
Explanation: querySelector returns the first match of any selector group.
19. Which is fastest for selecting one element by ID?
A. querySelector
B. getElementById
C. querySelectorAll
D. getElementsByClassName
Answer: B
Explanation: getElementById is optimized for ID lookup.
20. document.querySelectorAll("div")[0] gives:
A. First <div>
B. Last <div>
C. All divs
D. Error
Answer: A
Explanation: querySelectorAll returns NodeList, indexable like an array.
21. NodeList.forEach() works on:
A. all browsers ever made
B. modern browsers
C. no browsers
D. IE6 only
Answer: B
22. document.querySelector("p span") finds:
A. first span inside any p
B. all spans
C. all p elements
D. null always
Answer: A
23. getElementsByClassName returns:
A. static NodeList
B. live HTMLCollection
C. Array
D. object
Answer: B
24. If no element matches selector?
A. Error thrown
B. undefined
C. null for querySelector
D. empty string
Answer: C
25. querySelectorAll(".item").length returns:
A. number of items
B. array of items
C. string
D. null
Answer: A
