Final 25 questions · nested logic · switch chains · advanced conditions
Q76. Output?
let a = 0;
if (!a) console.log("Zero");
Answer: Zero

Explanation: 0 is falsy → !0 is true.
Q77. Predict result:
let x = 50;
if (x > 40)
  if (x < 60) console.log("Valid");
  else console.log("Out of range");
Answer: Valid

Explanation: Both conditions true.
Q78. Output?
let s = "0";
if (s) console.log("Truthy");
Answer: Truthy

Explanation: String "0" is non-empty.
Q79. Ternary result?
let marks = 42;
let res = marks>=40 ? "Pass" : "Fail";
console.log(res);
Answer: Pass

Explanation: 42≥40 true.
Q80. Switch output?
let mode="offline";
switch(mode){
  case "online": console.log("Web Class"); break;
  case "offline": console.log("Classroom"); break;
  default: console.log("N/A");
}
Answer: Classroom

Explanation: Matches "offline".
Q81. Output?
let a = undefined;
if (!a) console.log("Missing");
Answer: Missing

Explanation: undefined is falsy.
Q82. Which message prints?
let n = 30;
if (n>50) console.log("High");
else if (n>20) console.log("Medium");
else console.log("Low");
Answer: Medium

Explanation: n>20 is true.
Q83. Output?
let val = [];
if(val) console.log("Truthy array");
Answer: Truthy array

Explanation: Empty array is truthy.
Q84. Switch with no break?
let x=2;
switch(x){
  case 1: console.log("One");
  case 2: console.log("Two");
  case 3: console.log("Three");
}
Answer: Two
Three

Explanation: Fall-through starts at case 2.
Q85. Output?
let loggedIn=false;
console.log(loggedIn?"Welcome":"Please login");
Answer: Please login

Explanation: Condition false.
Q86. Output?
let c = "JS";
if(c && c[0]==="J") console.log("Match");
Answer: Match

Explanation: Truthy AND correct check.
Q87. Output?
let a="hello";
if(a.length>3) console.log("Long");
Answer: Long

Explanation: Length 5.
Q88. Multi-condition?
let s=70;
if(s>=90) console.log("A");
else if(s>=80) console.log("B");
else if(s>=70) console.log("C");
else console.log("D");
Answer: C

Explanation: 70≥70 is true.
Q89. Output?
let a = "";
if(!a) console.log("Empty string");
Answer: Empty string

Explanation: "" falsy → !"" true.
Q90. Predict output:
let num=15;
console.log(num%2===0?"Even":"Odd");
Answer: Odd

Explanation: 15%2=1.
Q91. Nested ternary:
let age=10;
let res = age>=18?"Adult":age>=13?"Teen":"Child";
console.log(res);
Answer: Child

Explanation: Both conditions false → last option.
Q92. switch logic?
let p="silver";
switch(p){
  case "gold": console.log("Premium"); break;
  case "silver":
  case "bronze": console.log("Standard"); break;
  default: console.log("Basic");
}
Answer: Standard

Explanation: silver falls in grouped case.
Q93. Output?
let val=null;
if(val===null) console.log("Null found");
Answer: Null found

Explanation: Strict null check.
Q94. Trick falsy:

Q95. Output?
let t=2;

Explanation: t===2.
Q96. Condition with &&:

Q97. Nested if:

  if(a<15) console.log("Range OK");

Explanation: Both conditions true.
Q98. Output?
let str="JS";

Q99. Output?

let attendance=78, paid=true;
