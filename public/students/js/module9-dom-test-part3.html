Module 9 — DOM Test Part 3
Creating, Appending & Removing Elements

25 Questions • DOM Manipulation • Click to Reveal Answers

1. Which method is used to create a new HTML element in JavaScript?

A. document.makeElement()
B. document.create()
C. document.createElement()
D. element.newNode()

Correct: C
Explanation: The method document.createElement() is used to create any new DOM element.
2. Which method adds a child node at the end of a parent element?

A. append
B. appendChild
C. addChild
D. insertLast

Correct: B
Explanation: appendChild() always appends a single node at the end.
3. What is the main difference between append and appendChild?

A. append can add multiple nodes or text
B. appendChild only adds text
C. append deletes previous children
D. appendChild supports multiple children at once

Correct: A
Explanation: append() can insert text + multiple nodes. appendChild() only accepts one node.
4. What is the output of this? 
const li = document.createElement("li");
li.textContent = "JS";
console.log(li);

A. <li></li>
B. li object
C. "JS"
D. <li>JS</li>

Correct: B
Explanation: Console shows an element object, not rendered HTML.
5. Which method removes an element from the DOM (modern way)?

A. removeNode()
B. delete()
C. element.remove()
D. removeElement()

Correct: C
Explanation: element.remove() is the modern removal method.
6. What does this add? 
list.append("New Item");

A. A new li element
B. A text node
C. Nothing
D. It throws an error

Correct: B
Explanation: Passing a string creates a text node.
7. Which inserts an item at the START of a list?

A. appendChild
B. append
C. prepend
D. insertBefore

Correct: C
Explanation: prepend() inserts content at the top.
8. Output? 
const box = document.createElement("div");
box.innerHTML = "<span>Hi</span>";
console.log(box.textContent);

A. "<span>Hi</span>"
B. "Hi"
C. undefined
D. "<div>Hi</div>"

Correct: B
Explanation: textContent returns only plain text inside elements.
9. Which creates a new <p> element? 
let p = ______ ;

A. document.make("p")
B. document.createElement("p")
C. new Element("p")
D. create.p()

item.removeChild(item.firstChild);

A. Removes the first element
B. Removes the last element
C. Removes the first child node
D. Deletes entire item

A. You cannot remove created elements
