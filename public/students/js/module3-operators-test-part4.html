Q76. Evaluate:
3 + 4 + "5"
Answer: "75"

Explanation: First 3+4=7 (number), then 7 + "5" → string.
Q77. Evaluate:
"5" + 3 + 4
Answer: "534"

Explanation: String first → whole chain becomes string.
Q78. Output:
console.log(10 > 5 > 1);
Answer: false

Explanation: (10 > 5) → true → true > 1 → false.
Q79. What is:
(10 > 5) && (5 > 3)?
Answer: true

Explanation: Both comparisons are true.
Q80. What is:
(10 > 5) && (3 > 5)?
Answer: false

Explanation: Second condition is false.
Q81. Evaluate:
true || false && false
Answer: true

Explanation: && runs first → false && false = false → true || false = true.
Q82. Evaluate:
false && true || true
Answer: true

Explanation: false && true = false → false || true = true.
Q83. Evaluate:
0 && "JS"
Answer: 0

Explanation: AND returns first falsy value.
