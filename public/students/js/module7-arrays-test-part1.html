Q1 ¬∑ Concept Single correct
Which of the following best describes a JavaScript array?
A) A fixed-size collection of only numbers
B) An ordered list that can store multiple values of any type
C) A key‚Äìvalue pair collection like an object
D) A special variable that can store only strings

Correct Answer: B

Explanation: In JavaScript, an array is an ordered list that can store multiple values of any type (numbers, strings, objects, etc.) in a single variable. It is not fixed-size and is different from objects and maps.

Q2 ¬∑ Output Indexing
What will be printed in the console?
let nums = [10, 20, 30, 40];
console.log(nums[2]);
A) 10
B) 20
C) 30
D) 40

Correct Answer: C

Explanation: Arrays are 0-indexed. nums[0] is 10, nums[1] is 20, nums[2] is 30, nums[3] is 40.

Q3 ¬∑ Concept length property
If let arr = [1, 2, 3, 4, 5]; what is the value of arr.length?
A) 4
B) 5
C) 6
D) Undefined

Correct Answer: B

Explanation: length returns the number of elements in the array. Here there are 5 elements, so arr.length === 5.

Q4 ¬∑ Output push()
What will this code output?
let fruits = ["Apple", "Banana"];
fruits.push("Mango");
console.log(fruits.length);
A) 1
B) 2
C) 3
D) 4

Correct Answer: C

Explanation: Initially 2 elements (Apple, Banana). push("Mango") adds one more at the end, so length becomes 3.

Q5 ¬∑ Concept Core methods
Which method removes the last element from an array and returns it?
A) shift()
B) unshift()
C) pop()
D) splice()

Correct Answer: C

Explanation: pop() removes and returns the last element. shift() works at the beginning, unshift() adds at the beginning, splice() can remove/insert at any index.

Q6 ¬∑ Output shift()
What is logged to the console?
let arr = [5, 10, 15];
let x = arr.shift();
console.log(x, arr);
A) 5 [10, 15]
B) 10 [5, 15]
C) 15 [5, 10]
D) 5 [5, 10, 15]

Correct Answer: A

Explanation: shift() removes and returns the first element. So x = 5 and array becomes [10, 15].

Q7 ¬∑ Concept Indexing
For let names = ["A", "B", "C", "D"]; which index gives the last element?
A) names[3]
B) names[4]
C) names[names.length]
D) names[-1]

Correct Answer: A

Explanation: There are 4 elements, indices 0‚Äì3. Last element is at index length - 1 ‚Üí 4 - 1 = 3.

Q8 ¬∑ Output Sparse array
What will be printed?
let arr = [];
arr[2] = "JS";
console.log(arr.length);
A) 0
B) 1
C) 2
D) 3

Correct Answer: D

Explanation: Setting index 2 creates a sparse array: indices 0 and 1 become empty (undefined). Length is highest index + 1 ‚áí 2 + 1 = 3.

Q9 ¬∑ Concept typeof
What is the result of typeof [] in JavaScript?
A) "array"
B) "object"
C) "list"
D) "undefined"

Correct Answer: B

Explanation: Due to JavaScript's design, typeof [] is "object". Arrays are special kind of objects; use Array.isArray() to check if something is an array.

Q10 ¬∑ Output unshift()
What does the following log?
let items = ["Pen", "Pencil"];
items.unshift("Eraser");
console.log(items);
A) ["Pen", "Pencil", "Eraser"]
B) ["Eraser", "Pen", "Pencil"]
C) ["Eraser", "Pencil"]
D) ["Pen", "Eraser", "Pencil"]

Correct Answer: B

Explanation: unshift() adds an element at the start of the array. So new array is ["Eraser","Pen","Pencil"].

Q11 ¬∑ Concept MCQ

Q12 ¬∑ Output Index out of range

Q13 ¬∑ Small Code Counting
Which option correctly prints all elements of arr one by one? 
let arr = [10, 20, 30];
A) for (let i = 1; i <= arr.length; i++) console.log(arr[i]);
B) for (let i = 0; i < arr.length; i++) console.log(arr[i]);
C) for (let i in arr) console.log(i);
D) console.log(arr.length());

Correct Answer: B

Explanation: Correct loop: start index 0, go while i < length, and print arr[i]. Option A starts from 1 and misses the first element and accesses undefined at the end.
