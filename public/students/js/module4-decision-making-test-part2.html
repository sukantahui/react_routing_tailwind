Part 2 (Q26–50)

else-if ladders · nested if · logical checks · multi-way branching
Q26. What prints?
let age = 20;
if (age > 18) console.log("Adult");
Answer: Adult

Explanation: age > 18 is true.
Q27. Output?
let score = 40;
if (score >= 50) console.log("Pass"); else console.log("Fail");
Answer: Fail

Explanation: 40 ≥ 50 is false.
Q28. Output of ladder?
let m = 72;
if (m >= 90) console.log("A+");
else if (m >= 80) console.log("A");
else if (m >= 70) console.log("B");
else console.log("C");
Answer: B

Explanation: First true condition is m ≥ 70.
Q29. Output?
let temp = 15;
if (temp > 30) console.log("Hot");
else if (temp > 20) console.log("Warm");
else console.log("Cold");
Answer: Cold

Explanation: 15 is not >20.
Q30. What prints?
let x = 5;
if (x > 10) console.log("A");
else if (x > 7) console.log("B");
else console.log("C");
Answer: C

Explanation: none of the if/else-if conditions are true.
Q31. Predict output:
let n = 0;
if (n) console.log("Non-zero"); else console.log("Zero");
Answer: Zero

Explanation: 0 is falsy.
Q32. What will print?
if ("JS") console.log("Works");
Answer: Works

Explanation: Non-empty string is truthy.
Q33. What is printed?
let a = -5;
if (a) console.log("True");
Answer: True

Explanation: All non-zero numbers are truthy (including negative).
Q34. Output?
let u = undefined;
if (!u) console.log("Empty");
Answer: Empty

Explanation: undefined is falsy → !undefined = true.
Q35. Ternary output?
(10 > 5) ? console.log("YES") : console.log("NO");
Answer: YES

Explanation: Condition is true.
Q36. Ternary evaluation:
(3 === "3") ? "Match" : "Mismatch"
Answer: Mismatch

Explanation: strict mismatch.
Q37. Output?
let loggedIn = false;
console.log(loggedIn ? "Welcome" : "Login required");
Answer: Login required

Explanation: false → second value executes.
Q38. Switch output:
let day = 2;
switch(day){
  case 1: console.log("Mon"); break;
  case 2: console.log("Tue"); break;
  default: console.log("Other");
}
Answer: Tue

Explanation: case 2 matched.
Q39. Switch result?
let mode = "upi";
switch(mode){
  case "cash": console.log("Cash"); break;
  case "upi": console.log("Online"); break;
  default: console.log("Other");
}
Answer: Online

Explanation: mode matches "upi".
Q40. Jump through missing break?
let c = 1;
switch(c){
  case 1: console.log("A");
  case 2: console.log("B");
  break;
}
Answer: A
B

Explanation: Missing break causes fall-through.
Q41. Output?
let v = 3;
switch(v){
  case 1:
  case 2:
  case 3: console.log("Group"); break;
  default: console.log("Other");
}
Answer: Group

Q42. Nested if output?
let marks = 85;
let income = 200000;
if (marks >= 80) {
  if (income <= 300000) console.log("Eligible");
}
Answer: Eligible

Explanation: Both nested conditions true.
