Module 9 — DOM Test Part 2

Changing Content · textContent · innerHTML · style · classList

1. Which property is safest for setting plain text in an element?
A. innerHTML
B. outerHTML
C. textContent
D. innerText
Correct: C
textContent sets only text, prevents HTML injection, and is fastest.
2. Which property interprets HTML tags when inserted?
A. textContent
B. innerHTML
C. value
D. tagContent
Correct: B
innerHTML parses the inserted content as HTML.
3. What will this code output? title.textContent = "<b>Hello</b>";
A. It will display bold “Hello”
B. It will display <b>Hello</b> literally
C. It will throw error
D. It removes all HTML
Correct: B
textContent does NOT parse HTML — it prints as plain text.
4. Which is the correct way to change color using JS?
A. element.style.color = "red";
B. element.color = "red";
C. element.css("color","red")
D. element.style("color","red")
Correct: A
DOM styling uses element.style.propertyName.
5. Which is better for large style changes?
A. style property
B. classList
C. innerHTML
D. DOMParser
Correct: B
classList lets you toggle entire predefined CSS classes.
6. What does element.classList.toggle("active") do?
A. Replaces all classes
B. Removes all classes
C. Adds/removes “active” based on current state
D. Adds two classes
Correct: C
toggle() switches a class on/off.
7. What happens when you use innerHTML with user input?
A. Nothing changes
B. Very safe and recommended
C. Can cause XSS vulnerabilities
D. HTML parsing breaks
Correct: C
User content via innerHTML can inject malicious scripts.
8. What does this code do? elem.style.fontSize = "22px";
A. Adds a CSS class
B. Changes inline style font-size
C. Breaks the CSS
D. Removes font
Correct: B
It directly adds inline CSS to the element.
9. title.innerHTML = "<i>JS</i>" will result in:
A. Exactly <i>JS</i>
B. Italic “JS”
C. Error
D. No change
Correct: B
innerHTML parses JS correctly.
10. Which method removes *all* classes?
A. classList.clear()
B. element.className = ""
C. classList.reset()
D. removeAll()
Correct: B
Setting className = "" wipes all classes at once.
