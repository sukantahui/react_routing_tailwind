Q1
Output ‚Äî Nested loop with skip and break
Output ¬∑ Nested for

What is the output of the following code?

for (let i = 1; i <= 3; i++) {
  for (let j = 1; j <= 4; j++) {
    if (j === 2) continue;
    if (i === 3 && j === 3) break;
    console.log(i, j);
  }
}
Answer & Explanation

Output:

1 1
1 3
1 4
2 1
2 3
2 4
3 1

For each i, we loop j = 1‚Ä¶4. When j === 2, continue skips logging. When i === 3 and j === 3, break stops the inner loop only. So for i = 3 we log 3 1, skip j = 2, and at j = 3 we break ‚Äî no 3 3 or 3 4.

Q3
Output ‚Äî do‚Ä¶while with post-decrement
Output ¬∑ do‚Ä¶while

What does this code print?

let n = 3;

do {
  console.log("n =", n);
} while (n-- > 0);
Answer & Explanation
n = 3
n = 2
n = 1
n = 0

do‚Ä¶while runs the body before checking the condition. The condition uses n-- (post-decrement), so comparison happens with the old value, then n is reduced:

  • After first body: n = 3 printed, condition 3 > 0 ‚Üí true, then n = 2
  • Print 2 ‚Üí check 2 > 0 ‚Üí true, then n = 1
  • Print 1 ‚Üí check 1 > 0 ‚Üí true, then n = 0
  • Print 0 ‚Üí check 0 > 0 ‚Üí false, then n = -1 and stop.

Q14
Logic ‚Äî Checking for a prime number
Logic ¬∑ prime

Briefly describe a loop-based approach to check if n is prime (greater than 1 and divisible only by 1 and itself).

Answer & Explanation

A common approach:

let n = 29;
let isPrime = n > 1;

for (let i = 2; i * i <= n; i++) {
  if (n % i === 0) {
    isPrime = false;
    break;
  }
}

We try dividing n by all numbers from 2 to ‚àön. If any divisor is found, n is not prime. Using i * i <= n instead of i <= n makes the loop faster.

Q19
Logic ‚Äî Check palindrome number
Logic ¬∑ while

Explain how you can use a while loop to check whether a given integer n is a palindrome (reads same forwards and backwards).

Answer & Explanation
let n = 12321;
let original = n;
let reversed = 0;

while (n > 0) {
  let digit = n % 10;
  reversed = reversed * 10 + digit;
  n = Math.floor(n / 10);
}

let isPalindrome = (original === reversed);

The loop builds reversed by repeatedly taking the last digit of n. If the reversed number equals the original, it is a palindrome.

MCQ ‚Äî Infinite loop identification
MCQ ¬∑ while

Which of the following loops will cause an infinite loop? 

A)  let i = 0;
    while (i < 5) {
      console.log(i);
      i++;
    }

B)  let i = 0;
    while (i < 5) {
      console.log(i);
    }

      console.log(i);
    }
Answer & Explanation

Correct option: B

In B, i is never updated, so i < 5 is always true. A and C both increment i, so they stop after 5 iterations.

What does continue do in a for loop?

  3. Restarts the loop from the beginning.

Q25
MCQ ‚Äî Where does break apply?
MCQ ¬∑ break

In nested loops, a plain break statement (without label) will:

  1. Stop both inner and outer loops.
  2. Stop only the inner-most loop in which it appears.
  3. Skip the current iteration and continue with the next.
  4. Restart the outer loop from the beginning.
Answer & Explanation

Correct option: B

A normal break always exits the nearest (innermost) loop that contains it. The outer loop continues with its next iteration (if any).
