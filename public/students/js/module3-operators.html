3. Assignment & Compound Assignment Operators

The assignment operator (=) stores a value in a variable.

let x = 10;

Compound Assignment

x += 5;   // x = x + 5

x /= 4;   // x = x / 4
x %= 3;   // x = x % 3
Compound assignments make your code shorter and cleaner.

These return true or false.
