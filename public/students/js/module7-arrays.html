Step 1 ¬∑ Why Arrays?

1. Arrays ‚Äî Why Do We Need Them?

An array is an ordered list of values. Instead of using 10 different variables for 10 marks, you store them in a single array. Arrays are the foundation for storing and processing collections of data in JavaScript.

1.1 Creating Arrays

// Using array literal (recommended)
let marks = [78, 85, 92, 67, 88];

// Using Array constructor (less common for beginners)
let numbers = new Array(10, 20, 30);

1.2 Indexing & Length

let students = ["Amit", "Neha", "Rohit"];

console.log(students[0]); // "Amit"  (first element)
console.log(students[1]); // "Neha"
console.log(students[2]); // "Rohit"

console.log(students.length); // 3
Array indices start from 0, not 1. Last index is length - 1.
Step 2 ¬∑ Basic Operations

2. Core Array Methods: Add & Remove Elements

These methods let you add or remove elements from the beginning or end of the array.

2.1 push() ‚Äî Add at the End

let fruits = ["Apple", "Banana"];

fruits.push("Mango");
console.log(fruits); // ["Apple", "Banana", "Mango"]

2.2 pop() ‚Äî Remove from the End

let lastFruit = fruits.pop();
console.log(lastFruit); // "Mango"
console.log(fruits);    // ["Apple", "Banana"]

2.3 unshift() ‚Äî Add at the Beginning

fruits.unshift("Orange");
console.log(fruits); // ["Orange", "Apple", "Banana"]

2.4 shift() ‚Äî Remove from the Beginning

let firstFruit = fruits.shift();
console.log(firstFruit); // "Orange"
console.log(fruits);     // ["Apple", "Banana"]
push/pop work at the end (fast); unshift/shift work at the start (slower for big arrays, but fine for learning & small projects).
Step 3 ¬∑ splice vs slice

3. splice() & slice() ‚Äî Editing & Copying

3.1 splice() ‚Äî Change the Original Array

Syntax: array.splice(startIndex, deleteCount, ...itemsToAdd)

let numbers = [10, 20, 30, 40, 50];

// Remove 2 elements from index 1
let removed = numbers.splice(1, 2);
console.log(numbers); // [10, 40, 50]
console.log(removed); // [20, 30]

// Insert elements without removing
numbers.splice(1, 0, 25, 35);
console.log(numbers); // [10, 25, 35, 40, 50]

3.2 slice() ‚Äî Copy Part of the Array

Syntax: array.slice(startIndex, endIndex) (end index is not included).

let marks = [60, 70, 80, 90];

let topTwo = marks.slice(2);      // from index 2 to end
console.log(topTwo);              // [80, 90]

let mid = marks.slice(1, 3);      // index 1 and 2
console.log(mid);                 // [70, 80]
splice modifies the original array.
slice returns a new array, leaving the original unchanged.
Step 4 ¬∑ Iteration

4. Iterating Over Arrays: Loops & forEach

Iteration means ‚Äúgoing through every element one by one‚Äù. We can do this with classic loops or with array helper methods.

4.1 Traditional for Loop

let students = ["Amit", "Neha", "Sourav"];

for (let i = 0; i < students.length; i++) {
  console.log(i, students[i]);
}

4.2 for...of ‚Äî Simpler Iteration

for (let name of students) {
  console.log(name);
}

4.3 forEach() ‚Äî Array Method for Iteration

students.forEach(function(name, index) {
  console.log(index, name);
});

// Arrow function style
students.forEach((name, index) => {
  console.log(`${index}: ${name}`);
});
forEach is great for printing or doing side effects. It does not return a new array ‚Äî it always returns undefined.
Step 5 ¬∑ Transformation

5. Transforming Arrays: map, filter, reduce

These are ‚Äúhigher-order‚Äù methods ‚Äî they take a function as an argument and produce new results from existing arrays. This is very important for modern JavaScript.

5.1 map() ‚Äî Transform Each Value

Creates a new array with the results of applying a function to every element.

let prices = [100, 200, 300];

// Add 18% GST to every price
let pricesWithGST = prices.map(price => price * 1.18);

console.log(pricesWithGST); // [118, 236, 354]

5.2 filter() ‚Äî Keep Some Values

Creates a new array with only the elements that match a condition.

let marks = [35, 55, 68, 92, 40];

// Keep only passing marks (>= 40)
let passed = marks.filter(m => m >= 40);
console.log(passed); // [55, 68, 92, 40]

5.3 reduce() ‚Äî Combine to a Single Value

Combines all elements into one result (sum, product, etc.).

let total = marks.reduce((accumulator, current) => {
  return accumulator + current;
}, 0);  // 0 is the initial value

console.log("Total marks:", total);
A very common pattern:
  • filter ‚Üí narrow down data
  • map ‚Üí transform each item
  • reduce ‚Üí summarise everything
Step 6 ¬∑ Search & Check

6. Searching & Checking in Arrays

6.1 indexOf() ‚Äî Find Position

let cities = ["Kolkata", "Delhi", "Mumbai"];

console.log(cities.indexOf("Kolkata")); // 0
console.log(cities.indexOf("Pune"));    // -1 (not found)

6.2 includes() ‚Äî Is It Present?

console.log(cities.includes("Mumbai")); // true
console.log(cities.includes("Pune"));   // false

6.3 find() ‚Äî Find First Matching Element

let students = [
  { name: "Amit", marks: 80 },
  { name: "Neha", marks: 92 },
  { name: "Rohit", marks: 65 }
];

let topper = students.find(s => s.marks > 90);

console.log(topper); // { name: "Neha", marks: 92 }
includes is great for simple arrays of primitive values (numbers/strings). find is powerful for arrays of objects.
Step 7 ¬∑ Set & Map

7. Intro to Set & Map

Set and Map are modern data structures in JavaScript: Set for unique values, Map for key‚Äìvalue pairs.

7.1 Set ‚Äî Collection of Unique Values

let numbers = [1, 2, 2, 3, 4, 4, 5];

let uniqueSet = new Set(numbers);
console.log(uniqueSet);           // Set(5) {1, 2, 3, 4, 5}

// Convert Set back to array
let uniqueArray = [...uniqueSet];
console.log(uniqueArray);         // [1, 2, 3, 4, 5]

7.2 Map ‚Äî Key‚ÄìValue Storage

let fees = new Map();

fees.set("Amit", 5000);
fees.set("Neha", 6000);

console.log(fees.get("Neha")); // 6000
console.log(fees.has("Amit")); // true

fees.delete("Amit");
console.log(fees.has("Amit")); // false
For teaching: Array ‚Üí simple ordered list Set ‚Üí unique list Map ‚Üí ‚Äúdictionary‚Äù of key ‚Üí value

Quick Reference ‚Äî Array Methods

  • Adding: push(), unshift()
  • Removing: pop(), shift(), splice()
  • Copy part: slice()
  • Loop: for, for...of, forEach()
  • Transform: map(), filter(), reduce()
  • Search: indexOf(), includes(), find()
