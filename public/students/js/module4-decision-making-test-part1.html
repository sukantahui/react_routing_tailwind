Module 4 — Decision Making
Test Part 1 (Q1–25)

if · else · else-if · nested logic · switch · truthy/falsy
Q1. What will this print?
if (true) console.log("Hello");
Answer: Hello

Explanation: Condition is true, so the statement executes.
Q2. Output?
if (false) console.log("A"); else console.log("B");
Answer: B

Explanation: false makes if skip, else runs.
Q3. What is printed?
let x=10;
if (x>5) console.log("Yes");
Answer: Yes

Explanation: x>5 is true.
Q4. Output?
if (0) console.log("Hi"); else console.log("Bye");
Answer: Bye

Explanation: 0 is falsy.
Q5. What logs?
if ("") console.log("X"); else console.log("Y");
Answer: Y

Explanation: Empty string is falsy.
Q6. What prints?
if ("hello") console.log("Run");
Answer: Run

Explanation: Non-empty strings are truthy.
Q7. Predict output:
let a = 5;
if (a===5) console.log("OK");
Answer: OK

Explanation: Strict equality matches exactly.
Q8. What logs?
let x;
if (x) console.log("A"); else console.log("B");
Answer: B

Explanation: undefined is falsy.
Q9. Output?
if (null) console.log("1"); else console.log("2");
Answer: 2

Explanation: null is falsy.
Q10. Output?
if (NaN) console.log("T"); else console.log("F");
Answer: F

Explanation: NaN is falsy.
Q11. Evaluate:
if (5=="5") console.log("Same");
Answer: Same

Explanation: == converts types (not recommended).
Q12. What logs?
if (5==="5") console.log("Yes"); else console.log("No");
Answer: No

Explanation: Types differ in strict comparison.
Q13. Output?
if (true && false) console.log("A"); else console.log("B");
Answer: B

Explanation: true && false = false.
Q14. Output?
if (false || true) console.log("Hi");
Answer: Hi

Explanation: OR returns true if any operand true.
Q15. Evaluate:
if (!false) console.log("Run");
Answer: Run

Explanation: !false = true.
Q16. What will print?
let a=10;
if(a>5 && a<20) console.log("OK");
Answer: OK

Explanation: Both conditions true.
Q17. Output?
let x=5;
if(x==5) console.log("Match");
Answer: Match

Explanation: Loose match returns true.
Q18. Output?
if("0") console.log("Yes");
Answer: Yes

Explanation: "0" is non-empty → truthy.
Q19. Predict:
if (5>10) console.log("X"); else console.log("Y");
Answer: Y

Explanation: 5>10 is false.
Q20. Output?
if (10>=10) console.log("Equal");
Answer: Equal

Explanation: >= includes equality.
Q21. What prints?
let x=2;
if(x%2===0) console.log("Even");
Answer: Even

Explanation: 2%2 = 0.
Q22. Output?
let a=7;
if(a%2===1) console.log("Odd");
Answer: Odd

Explanation: 7%2=1.
Q23. Evaluate:
if(false || false) console.log("Run"); else console.log("Skip");
Answer: Skip

Explanation: false || false = false.
Q24. Output?
if(!"") console.log("Empty");
Answer: Empty

Explanation: "" is falsy → !"": true.
Q25. What prints?
if(5!==6) console.log("True");
Answer: True

Explanation: 5!==6 → true.
