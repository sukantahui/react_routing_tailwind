Q18 Â· Coding
Write a function calculateAverage(arr) that returns the average of numeric values in the array using reduce().

Example Solution:

function calculateAverage(arr) {
  if (arr.length === 0) return 0;
  const total = arr.reduce((sum, n) => sum + n, 0);
  return total / arr.length;
}

// Example:

Explanation: reduce() gives total; divide by length for average. Always handle the empty-array case safely.
