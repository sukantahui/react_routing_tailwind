Step 1 · Foundations

1. What Is Node.js?

Node.js allows JavaScript to run *outside the browser* using the V8 engine. It is commonly used for backend development, automation scripts, tooling, and servers.

  • Runs JS in terminal (server-side).
  • Non-blocking & asynchronous by design.
  • Powers backend frameworks like Express.js.
  • Used heavily for automation (build tools, bundlers).
$ node --version
v20.8.0
If you can run node file.js, you are running JavaScript without a browser.
Step 2 · First Script

2. Running JavaScript Files with Node

Node.js executes JavaScript files directly using the command: 

$ node app.js

2.1 Example: app.js

// app.js
console.log("Hello from Node.js!");

Run it:

$ node app.js
Hello from Node.js!

2.2 Reading Command-Line Arguments

// args.js
console.log("Arguments:", process.argv);

$ node args.js love javascript
Arguments: ["/usr/bin/node","args.js","love","javascript"]
Step 3 · I/O Basics

3. Reading Input & Printing Output

3.1 Using readline

const readline = require("readline");

const rl = readline.createInterface({
  input: process.stdin,
  output: process.stdout
});

rl.question("Enter your name: ", (name) => {
  console.log("Hello", name);
  rl.close();
});
readline is commonly used for CLI apps (calculators, quizzes, small tools).
Step 4 · File System

4. Simple File Operations (Concept Level)

Node.js provides fs module for working with files.

4.1 Write a File

const fs = require("fs");

fs.writeFileSync("data.txt", "Hello World!");
console.log("File created.");

4.2 Read a File

const content = fs.readFileSync("data.txt", "utf8");
console.log(content);

4.3 Append to a File

fs.appendFileSync("data.txt", "\nNew Line Added");

4.4 Delete a File

fs.unlinkSync("data.txt");
Teach the idea — do not deep dive yet (full file handling comes in advanced backend modules).
Step 5 · Express Intro

5. Overview of Express.js & REST APIs

Express.js is a minimal and flexible Node.js web application framework that makes it easy to build REST APIs.

5.1 Installing Express (Concept)

npm install express

5.2 First Express Server

const express = require("express");
const app = express();

app.get("/", (req, res) => {
  res.send("Welcome to Coder & AccoTax API!");
});

app.listen(3000, () => {
  console.log("Server running on port 3000");
});

5.3 Simple REST Endpoints

app.get("/students", (req, res) => {
  res.json(["Amit","Riya","John"]);
});

app.post("/students", (req, res) => {
  res.json({ message: "Student added!" });
});
Do not go deep into Express now — just explain concepts: routes, GET/POST, JSON responses.
Summary

6. Summary & Practice Tasks

6.1 Summary

  • Node.js executes JS outside the browser using V8 engine.
  • Use node file.js to run scripts.
  • Command-line input via process.argv or readline.
  • fs module handles file operations.
  • Express.js builds REST APIs → simple GET/POST examples.

    • GET /
