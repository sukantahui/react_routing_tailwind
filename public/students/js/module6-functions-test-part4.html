Q1 ¬∑ Output Tracing ‚Äî Basic Return

Q9 ¬∑ MCQ ‚Äî Function Declaration

Q11 ¬∑ MCQ ‚Äî Arrow Functions

  • B) const prod = a, b => a * b;
  • C) const prod = (a, b) => a * b;

Q14 ¬∑ MCQ ‚Äî Scope

Which variables are block scoped in JavaScript?

  • A) Only var
  • B) Only let

Q17 ¬∑ Coding Logic ‚Äî Fix the Bug

Q18 ¬∑ Coding Logic ‚Äî Using Rest

Q24 ¬∑ Advanced Logic ‚Äî Inner Function (Closure Idea)

Q25 ¬∑ Advanced Logic ‚Äî Function Returning Function

What gets logged?

  return function (n) {
    return n * factor;
  };

Answer:

20

makeMultiplier(2) returns a function that multiplies by 2 ‚Üí double(10) = 20. makeMultiplier(3) returns a function that multiplies by 3 ‚Üí triple(10) = 30. This is a classic closure pattern: inner functions remember factor.
