Q1 ¬∑ Concept (Entry/Exit Controlled) ‚Äî 1 mark
Which of the following is an exit-controlled loop in JavaScript?
  • A. for loop
  • B. while loop
  • C. do‚Ä¶while loop
  • D. Both for and while

Correct Answer: C

Explanation: In JavaScript, do‚Ä¶while is exit-controlled because the condition is checked after executing the loop body. Therefore, the body runs at least once. for and while are entry-controlled loops.

Q2 ¬∑ Output Tracing ‚Äî 2 marks
Consider the code: 
let i;

for (i = 1; i <= 10; i += 3) {
  if (i === 7) {
    break;
  }
  console.log(i);
}
(a) What will be the output on the console?
(b) What will be the final value stored in i after the loop terminates?

Correct Answer:
(a) Output:
1
4
(b) Final value of i is 7.

Explanation: Iterations:
‚Ä¢ Start: i = 1 ‚Üí condition 1 ‚â§ 10 true ‚Üí i === 7 false ‚Üí prints 1 ‚Üí i += 3 ‚Üí i = 4.
‚Ä¢ Next: i = 4 ‚Üí prints 4 ‚Üí i = 7.
‚Ä¢ Next: i = 7 ‚Üí condition true, i === 7 true ‚Üí break executed.
The loop exits immediately after hitting 7, so 7 is not printed, but it stays as the final value of i.

Explanation: Outer loop runs with i = 1, 2, 3. For each value of i, inner loop runs with j = 1, 2. So total iterations = 3 √ó 2 = 6, and each line prints current i and j.

Q4 ¬∑ Theory ‚Äî 2 marks
Differentiate between break and continue statements in the context of loops in JavaScript.

Suggested Points:

Explanation: ‚Ä¢ break: Immediately terminates the loop. Control moves to the first statement after the loop.
‚Ä¢ continue: Skips the current iteration of the loop and jumps to the next iteration (re-checking the condition in for/while).
‚Ä¢ break stops the loop completely; continue only skips one cycle.

Q5 ¬∑ Error Finding ‚Äî 2 marks
Find and correct the error(s) in the following loop so that it prints numbers from 5 down to 1: 
let i = 5;
while (i >= 1);
{
  console.log(i);
  i--;
}

Corrected Version:

let i = 5;
while (i >= 1) {
  console.log(i);
  i--;
}

Explanation: The error is the semicolon ; immediately after while (i >= 1). That semicolon ends the loop, making it an empty loop. The block in braces becomes a simple block that executes once. Removing the semicolon makes the block the correct loop body.

Q12 ¬∑ do‚Ä¶while vs while ‚Äî 2 marks
Write any two differences between while and do‚Ä¶while loops in JavaScript.

Suggested Points:

Explanation: 1. Condition check position:
‚Ä¢ while ‚Üí condition checked before executing the body (entry-controlled).
‚Ä¢ do‚Ä¶while ‚Üí condition checked after executing the body (exit-controlled).

2. Minimum executions:
‚Ä¢ while ‚Üí body may not execute even once if condition is false initially.
‚Ä¢ do‚Ä¶while ‚Üí body executes at least once, even if condition is false initially.

3. Syntax difference: do‚Ä¶while ends with a semicolon after the closing parenthesis of while(...);.

Q13 ¬∑ Infinite Loop Detection ‚Äî 2 marks

Q19 ¬∑ MCQ ‚Äî Truth about for(;;) ‚Äî 1 mark
What does the loop for(;;) represent in JavaScript?
  • A. A syntax error
  • B. A loop that runs exactly once
  • C. An infinite loop
  • D. A loop that runs only if a condition is passed inside

Explanation: for(;;) omits initialization, condition, and update. The missing condition is treated as true, so it is an infinite loop (unless break is used inside).
