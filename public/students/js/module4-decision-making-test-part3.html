Module 4 — Decision Making Test
Part 3 (Q51–75)

Advanced if-else · nested conditions · switch logic · tricky truthy/falsy
Q51. What prints?
let x = 5, y = 10;
if (x < y && y < 15) console.log("Valid");
else console.log("Invalid");
Answer: Valid

Explanation: Both conditions are true.
Q52. Output?
let a = 15;
if (a > 20) console.log("A");
else if (a > 10) console.log("B");
else console.log("C");
Answer: B

Explanation: a>10 is true, ladder stops here.
Q53. Predict output:
let n = -1;
if (n) console.log("Yes"); else console.log("No");
Answer: Yes

Explanation: Any non-zero number is truthy.
Q54. Output?
let s = " ";
if (s) console.log("True");
Answer: True

Explanation: " " is non-empty → truthy.
Q55. Which prints?
let m = 65;
if (m >= 80) console.log("A");
else if (m >= 60) console.log("B");
else console.log("C");
Answer: B

Explanation: First true condition is m≥60.
Q56. Evaluate:
(false || true) ? "X" : "Y"
Answer: X

Explanation: Condition becomes true.
Q57. Nested if result?
let mark = 90;
let income = 600000;
if (mark >= 85) {
  if (income <= 500000) console.log("Scholarship");
  else console.log("No scholarship");
}
Answer: No scholarship

Explanation: Outer if true, inner if false.
Q58. Output?
let a = 3;
let b = 6;
if (a%3===0 && b%3===0) console.log("Both");
Answer: Both

Explanation: Both divisible by 3.
Q59. Trick output:
let x;
if (x = 1) console.log("Run");
Answer: Run

Explanation: Assignment → 1 (truthy).
Q60. Ternary output?
let age=25; console.log(age>18?"Allowed":"Denied");
Answer: Allowed

Explanation: Condition true.
Q61. Multi-ternary output?
let a = 75;
let result = a>=90 ? "A+" : a>=80 ? "A" : a>=70 ? "B" : "C";
console.log(result);
Answer: B

Explanation: Falls into a≥70.
Q62. Switch output?
let city = "kolkata";
switch(city){
  case "delhi": console.log("North"); break;
  case "kolkata": console.log("East"); break;
  default: console.log("Other");
}
Answer: East

Explanation: Case matches "kolkata".
Q63. Output?
let t=4;
switch(t){
  case 1:
  case 2:
  case 3: console.log("Low");
          break;
  case 4:
  case 5: console.log("High");
          break;
}
Answer: High

Explanation: 4 matches second group.
Q64. What prints?
let str = "";
if (str) console.log("Yes"); else console.log("No");
Answer: No

Explanation: "" is falsy.
Q65. Predict:
let q=null;
if(!q) console.log("Empty");
Answer: Empty

Explanation: null is falsy → !null true.
Q66. Output?
let a=10,b=5;
if (a>b && b>0) console.log("OK");
Answer: OK

Explanation: Both conditions true.
Q67. Output?
let rain=false, cloudy=true;
if(rain || cloudy) console.log("Take umbrella");
Answer: Take umbrella

Explanation: OR: at least one true.
Q68. Output?
let x = 10;
if (x<5) console.log("A");
else if (x<15) console.log("B");
else console.log("C");
Answer: B

Explanation: x<15 true.
Q69. Switch + default?
let fruit="apple";
switch(fruit){
  case "banana": console.log("Yellow"); break;
  default: console.log("Unknown");
}
Answer: Unknown

Explanation: apple doesn't match case.
Q70. Output?
let role="guest";
console.log(role==="admin"?"Full":"Limited");
Answer: Limited

Explanation: guest !== admin.
Q71. Output?
let a=2,b=4;
if(a===2){
  if(b===4) console.log("Match");
}
Answer: Match

Explanation: Both conditions true.
Q72. Output?
let val = "JS";
if(val && val.length>1) console.log("Valid");
Answer: Valid

Explanation: Truthy AND length>1.
Q73. Confusing falsy?
let num = NaN;
if(!num) console.log("NaN detected");
Answer: NaN detected

Explanation: NaN is falsy.
Q74. Output?
let c=12;
if(c%2===0 && c%3===0) console.log("Both");
Answer: Both

Explanation: 12 divisible by both 2 and 3.
Q75. Multi-branch output?
let temp=29;
if(temp>35) console.log("Hot");
else if(temp>25) console.log("Warm");
else console.log("Cool");
Answer: Warm

Explanation: 29>25 is true.
