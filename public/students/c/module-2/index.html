Module 2 — Variables & Data Types

Variables, Constants & Data Types

Learn how C stores data in memory, how variables work, and how data types define memory size and behavior.

0. Tokens in C

A token is the smallest meaningful unit in a C program — similar to a word in a sentence. The compiler breaks your source code into tokens before understanding it.

Common token categories in C:

  • 1. Keywords – Reserved words that have a special meaning in C and cannot be used as identifiers.
    Examples: int, float, if, else, return, for
  • 2. Identifiers – Names given by the programmer to variables, functions, arrays, structures, etc.
    Examples: age, sum, totalMarks, calculateAverage
  • 3. Constants / Literals – Fixed values that do not change during program execution.
    Examples:
    • Integer literal: 10, -25
    • Floating literal: 3.14, 0.001
    • Character literal: 'A', '9'
    • String literal: "Hello", "Coder & AccoTax"
  • 4. Operators – Symbols that perform operations on variables and values.
    Examples: +, -, *, /, %, ==, &&, ||
  • 5. Punctuators (Special Symbols) – Symbols that define structure and separation in C code.
    Examples: ;, ,, (), {}, [], #

Example: Tokens inside a C statement

int total = price + 10;

Breakdown:
int → keyword
total → identifier (variable name)
= → assignment operator
price → identifier
+ → arithmetic operator
10 → integer literal (constant)
; → punctuator (statement terminator)

Understanding tokens first makes it easier to learn variables, data types, operators and full C syntax in the next sections.

1. What is a Variable?

A variable is a named memory location that can store data.

  • Variables allow storing values that can change during program execution.
  • Each variable has a name, type, and memory address.
  • C is statically typed — type is declared before use.
// Example
int age = 20;
float price = 99.5;
char grade = 'A';

2. Rules for Naming Variables

C follows strict rules for identifiers.

  • Can contain letters, digits, and underscore (_).
  • Cannot start with a digit.
  • No spaces or special characters allowed.
  • C is case-sensitive: Age and age are different.
  • Keywords cannot be used (e.g., int, float, return).

3. Data Types in C

Used to declare the type and size of data.

  • int – Integer values
  • float – Single-precision decimal
  • double – Double-precision decimal
  • char – Single character
// Example
int marks = 85;
float height = 5.11;
double salary = 50500.75;
char letter = 'Z';

4. Constants in C

Values that do not change during execution.

  • Defined using const keyword.
  • Or using #define macro.
// Using const
const int max = 100;

// Using #define
#define PI 3.14

5. Basic Input & Output

Using printf() and scanf().

  • printf() is used to display output.
  • scanf() is used for taking input from user.
int age;
printf("Enter your age: ");
scanf("%d", &age);
printf("You entered: %d", age);

6. Format Specifiers in C

Format specifiers tell printf() and scanf() the type of data to print or read.

  • C does not automatically know the type of variable during input/output.
  • Format specifiers act as instructions for how the data should be interpreted.
  • They ensure correct reading and printing of memory values.
// Examples
printf("%d", age);       // prints integer
printf("%f", height);    // prints float
printf("%c", grade);     // prints character
printf("%s", name);      // prints string

scanf("%d", &age);       // reads integer
scanf("%f", &height);    // reads float

Common Format Specifiers

  • %d – Integer (int)
  • %f – Floating number (float)
  • %lf – Double
  • %c – Character
  • %s – String (char array)
  • %u – Unsigned integer
  • %x / %X – Hexadecimal output
  • %p – Pointer (address)
// Example: formatted output
int a = 10;
float b = 5.6732;

printf("A = %d, B = %.2f", a, b);
// Output: A = 10, B = 5.67

Note: Format specifiers are essential because C treats each data type differently in memory. Without them, printf() and scanf() cannot correctly interpret the data.
