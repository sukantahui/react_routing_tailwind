Module 16 — Data Structures

Data Structures with C

Use C to build classic data structures like Stack, Queue, and Linked List, and implement basic searching and sorting algorithms.

1. Introduction to Data Structures

Data structures organize and store data efficiently.

  • Stack – Last In First Out (LIFO)
  • Queue – First In First Out (FIFO)
  • Linked List – Dynamic node-based structure
  • Used in OS, compilers, DBMS, networks, games, etc.

2. Stack (Using Array)

Stack supports two main operations: push and pop.

#define SIZE 5
int stack[SIZE];
int top = -1;

void push(int x) {
    if(top == SIZE - 1) {
        printf("Stack Overflow\n");
        return;
    }
    stack[++top] = x;
}

int pop() {
    if(top == -1) {
        printf("Stack Underflow\n");
        return -1;
    }
    return stack[top--];
}

void displayStack() {
    for(int i = top; i >= 0; i--) {
        printf("%d ", stack[i]);
    }
}

3. Queue (Using Array)

Queue supports enqueue (insert) and dequeue (remove).

#define QSIZE 5
int queue[QSIZE];
int front = 0;
int rear = -1;
int countQ = 0;

void enqueue(int x) {
    if(countQ == QSIZE) {
        printf("Queue Overflow\n");
        return;
    }
    rear = (rear + 1) % QSIZE;
    queue[rear] = x;
    countQ++;
}

int dequeue() {
    if(countQ == 0) {
        printf("Queue Underflow\n");
        return -1;
    }
    int val = queue[front];
    front = (front + 1) % QSIZE;
    countQ--;
    return val;
}

void displayQueue() {
    int i = front;
    int c = countQ;
    while(c--) {
        printf("%d ", queue[i]);
        i = (i + 1) % QSIZE;
    }
}

4. Singly Linked List

Nodes connected via pointers, dynamically allocated.

struct Node {
    int data;
    struct Node *next;
};

struct Node *head = NULL;

void insertAtBeginning(int x) {
    struct Node *newNode =
        (struct Node*) malloc(sizeof(struct Node));
    newNode->data = x;
    newNode->next = head;
    head = newNode;
}

void displayList() {
    struct Node *temp = head;
    while(temp != NULL) {
        printf("%d -> ", temp->data);
        temp = temp->next;
    }
    printf("NULL\n");
}

5. Linear Search

Search each element one by one.

int linearSearch(int arr[], int n, int key) {
    for(int i = 0; i < n; i++) {
        if(arr[i] == key)
            return i;  // index
    }
    return -1; // not found
}

6. Bubble Sort

Simple comparison-based sorting.

void bubbleSort(int arr[], int n) {
    int temp;
    for(int i = 0; i < n-1; i++) {
        for(int j = 0; j < n-1-i; j++) {
            if(arr[j] > arr[j+1]) {
                temp = arr[j];
                arr[j] = arr[j+1];
                arr[j+1] = temp;
            }
        }
    }
}

7. Summary & Usage

  • Use Stack for undo operations, expression evaluation, recursion.
  • Use Queue for scheduling, printer spooling, buffering.
  • Use Linked List for dynamic collections with frequent insertion/deletion.
  • Use Searching & Sorting to manage and retrieve data efficiently.
