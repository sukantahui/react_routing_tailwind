A. Multiple Choice Questions (MCQs)

1. Which statement is used for multi-way branching?
A) if
B) else-if ladder
C) switch
D) break
2. What is the default value of default section in switch?
A) It must be written first
B) It runs only if case 0 matches
C) It executes if no case matches
D) It is optional but must be empty
3. Which of these can replace multiple nested if-else?
A) break
B) switch
C) continue
D) goto
4. Which statement forces exit from a loop or switch?
A) end
B) return
C) break
D) stop
5. What does continue do in loops?
A) Stops loop completely
B) Skips current iteration
C) Ends program
D) Repeats entire loop twice
