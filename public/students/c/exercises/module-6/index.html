Arrays — Practice

Practice problems on 1D arrays, 2D arrays, initialization, traversal, searching, and basic array operations used in C programming.

A. Multiple Choice Questions

1. Array indexing in C starts from:
A) -1
B) 0
C) 1
D) Any value
2. What is the correct declaration of a 1D array?
A) int arr[];
B) int arr[10];
C) int [10] arr;
D) array int arr;
3. Which function is commonly used to read array input?
A) input()
B) cout<<
C) scanf()
D) read()
4. What is the size of int arr[10] (assuming int = 4 bytes)?
A) 10 bytes
B) 20 bytes
C) 40 bytes
D) Depends on compiler
5. A 2D array is stored in:
A) Column-major order
B) Tree structure
C) Contiguous memory
D) Linked list memory

C. Coding Practice

  • 1. Write a program to store 10 integers in an array and print them.
  • 2. Write a program to find the largest number in an array of 10 integers.
  • 3. Write a program to count even and odd numbers in an array.
  • 4. Write a program to search for an element in an array (linear search).
  • 5. Write a program to reverse a 1D array.
  • 6. Write a program to add two 2D matrices of size 2×2.
  • 7. Write a program to read a 3×3 matrix and print the sum of all elements.
  • 8. Write a program to calculate row-wise and column-wise sums of a 2D matrix.
  • 9. Write a program to find the maximum element in each row of a 2D array.
  • 10. Write a program to print the diagonal elements of a 3×3 matrix.
