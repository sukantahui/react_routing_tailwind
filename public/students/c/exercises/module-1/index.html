Introduction to C — Practice

MCQs, output questions, and coding practice based on Module 1: What is C, history, features, structure of a C program, and first program.

A. Multiple Choice Questions

Choose the correct option.

1. C is often called a ______ language.
A) High-level
B) Middle-level
C) Low-level
D) Machine language
2. The creator of C language is:
A) James Gosling
B) Bjarne Stroustrup
C) Dennis Ritchie
D) Guido van Rossum
3. Which of the following is a valid C statement terminator?
A) Colon (:)
B) Semicolon (;)
C) Comma (,)
D) Dot (.)
4. The main() function in C is:
A) Optional
B) Must appear twice
C) The entry point of program
D) Only used in C++
5. Which is used to write output on the screen in C?
A) echo()
B) cout<<
C) printf()
D) display()

B. Output Based Questions

Write the exact output of the following programs.

1.
#include <stdio.h>
int main() {
    printf("Hello C");
    return 0;
}
2.
#include <stdio.h>
int main() {
    printf("Coder ");
    printf("& AccoTax");
    return 0;
}
3.
#include <stdio.h>
int main() {
    printf("Line1\nLine2\nLine3");
    return 0;
}

C. Coding Practice

Write C programs for the following tasks.

  • 1. Write a program to print “Welcome to C Programming” on the screen.
  • 2. Write a program to print your name, class and city in separate lines.
  • 3. Write a program to display the text:
    Coder & AccoTax — C Programming Course
  • 4. Write a program to print a simple box using * characters.
  • 5. Write a program to display your school/college name 5 times.
