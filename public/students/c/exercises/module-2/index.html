Variables, Constants & Data Types — Practice

Practice questions on identifiers, keywords, data types, constants and formatted input/output in C.

A. Multiple Choice Questions

Choose the correct option.

1. Which of the following is a valid C identifier?
A) 2value
B) roll-no
C) total_marks
D) float
2. Which keyword is used to define a constant in C?
A) let
B) const
C) static
D) final
3. What is the correct format specifier for float in printf?
A) %d
B) %c
C) %f
D) %s
4. Which of these is a user-defined identifier?
A) int
B) char
C) main
D) printf
5. In C, char is generally stored using:
A) 1 byte
B) 2 bytes
C) 4 bytes
D) 8 bytes

B. Output Based Questions

1.
#include <stdio.h>
int main() {
    int a = 10;
    float b = 5.5;
    printf("%d %.1f", a, b);
    return 0;
}
2.
#include <stdio.h>
int main() {
    const int x = 5;
    printf("%d", x);
    // x = 7;  // what happens if we uncomment this?
    return 0;
}

C. Coding Practice

  • 1. Write a program to read an integer, a float and a character from the user and display them in one line.
  • 2. Write a program to define a constant PI = 3.14 and calculate area of a circle for given radius.
  • 3. Write a program that reads marks (int) and percentage (float) and prints them formatted as:
    Marks: 450, Percentage: 90.00%

D. Answer Key (Quick Check)

MCQ Answers:
1. C   2. B   3. C   4. C   5. A
Output 1: 10 5.5
Output 2: prints 5; uncommenting assignment to x causes a compilation error (modifying const).
