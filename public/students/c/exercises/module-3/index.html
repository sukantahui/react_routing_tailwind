Module 3 — Exercise Set

Operators & Expressions — Practice

Test your understanding of arithmetic, relational, logical, assignment, and bitwise operators along with expression evaluation.

A. Multiple Choice Questions

1. The operator == is:
A) Assignment operator
B) Equality operator
C) Logical operator
D) Bitwise operator
2. What is the result of 5 / 2 in C (integer division)?
A) 2.5
B) 3
C) 2
D) 0
3. Which of the following is a logical operator?
A) &
B) &&
C) |
D) ^
4. The expression a += 5; is equivalent to:
A) a = a + 5;
B) a = a - 5;
C) a = 5;
D) a = a * 5;
5. Which has higher precedence?
A) * (multiplication)
B) + (addition)
C) = (assignment)
D) == (equality)

B. Output Based Questions

1.
int a = 5, b = 2;
printf("%d", a / b);
2.
int x = 5;
printf("%d %d", x++, ++x);
3.
int a = 4, b = 3;
printf("%d", a && b);

C. Coding Practice

  • 1. Write a program to read two integers and print their sum, difference, product and quotient.
  • 2. Write a program that reads an integer and prints whether it is even or odd using conditional operator ?:.
  • 3. Write a program to demonstrate all relational operators (<, >, <=, >=, ==, !=) between two numbers.

D. Answer Key

MCQ Answers:
1. B   2. C   3. B   4. A   5. A
Output 1: 2
Output 2: undefined behavior (sequence point issue)
Output 3: 1 (true, since both non-zero)
