A. Multiple Choice Questions

Choose the correct answer.

1. Which loop guarantees at least one execution?
A) for
B) while
C) do-while
D) nested loop
2. The loop that continues until a condition becomes false is:
A) switch loop
B) while loop
C) break loop
D) continue loop
3. Which of the following can cause an infinite loop?
A) Missing increment/decrement
B) Wrong condition
C) Always-true condition
D) All of these
4. The break statement:
A) Skips next iteration
B) Exits loop immediately
C) Repeats loop twice
D) Does nothing
5. Which loop is best for counting iterations?
A) if loop
B) while
C) do-while
D) for

B. Output Based Questions

1.
int i;
for(i = 1; i <= 5; i++)
    printf("%d ", i);
2.
int i = 1;
while(i < 5){
    printf("%d ", i);
    i++;
}
3.
int i = 5;
do{
    printf("%d ", i);
    i--;
}while(i >= 3);
4.
for(int i=1;i<=3;i++){
    for(int j=1;j<=2;j++){
        printf("%d%d ", i, j);
    }
}
