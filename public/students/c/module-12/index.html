Module 12 — File Handling

File Handling in C

Files allow storing and retrieving data permanently on disk. Learn how to open, close, read, write, and manage text & binary files in C.

1. What is File Handling?

Using files lets programs store data permanently.

  • Data stored in secondary memory
  • Persists even after the program ends
  • C supports text & binary files
  • All file operations use FILE * pointer
FILE *fp;   // file pointer

2. Opening & Closing Files

Use fopen() to open and fclose() to close.

FILE *fp;
fp = fopen("data.txt", "w");  // open for writing

if(fp == NULL){
    printf("File not found!");
}

fclose(fp);
  • "r" – read
  • "w" – write (creates new file)
  • "a" – append
  • "r+", "w+", "a+" – read/write

3. Writing to a File

FILE *fp = fopen("notes.txt", "w");
fprintf(fp, "Hello Students!");
fclose(fp);
    
FILE *fp = fopen("marks.txt", "w");

int m = 88;
fprintf(fp, "%d", m);

fclose(fp);

4. Reading from a File

FILE *fp = fopen("notes.txt", "r");
char text[100];

fgets(text, 100, fp);   // read a line
printf("%s", text);

fclose(fp);
    
FILE *fp = fopen("data.txt", "r");
int value;

fscanf(fp, "%d", &value);   // read a value
fclose(fp);

5. Character I/O Functions

fputc('A', fp);   // write char
char c = fgetc(fp); // read char

6. Random Access (fseek, ftell, rewind)

Jump to any file position.

fseek(fp, 0, SEEK_END);   // go to end
long size = ftell(fp);    // file size
rewind(fp);               // back to beginning

7. Binary File Handling

struct Student {
    int roll;
    char name[20];
};

FILE *fp = fopen("student.dat", "wb");

struct Student s = {1, "Riya"};
fwrite(&s, sizeof(s), 1, fp);

fclose(fp);
    
// Reading binary file
FILE *fp = fopen("student.dat", "rb");

struct Student s;
fread(&s, sizeof(s), 1, fp);

printf("%d %s", s.roll, s.name);

fclose(fp);

8. Common File Handling Errors

  • Opening a file with incorrect mode
  • Using uninitialized file pointer
  • Forgetting to close file
  • Assuming file exists when opening in r mode
  • Reading incorrect data type
