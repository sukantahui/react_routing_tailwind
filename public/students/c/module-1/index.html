Module 1 — Introduction to C

Introduction to C Programming

This module introduces the foundation of C—its history, features, structure, and the process of writing and compiling your first program.

1. What is C Programming?

A powerful, fast, and foundational programming language.

  • C is a general-purpose, procedural programming language.
  • Created by Dennis Ritchie at Bell Labs in 1972.
  • Forms the base of many modern languages (C++, Java, Python indirectly).
  • Used in OS development, embedded systems, system programming.

2. Key Features of C

Why C remains relevant even today.

  • Fast and efficient.
  • Low-level memory access.
  • Portable & machine independent.
  • Rich library of built-in functions.
  • Powerful pointer manipulation.

3. Structure of a C Program

Every C program follows a clean format.

  • Header files (e.g., #include <stdio.h>)
  • The main() function
  • Declarations & statements
  • Return statement

4. Writing Your First C Program

The famous Hello World program.

#include <stdio.h>

int main() {
    printf("Hello, World!");
    return 0;
}

5. Compilation & Execution Process

How the C code becomes an executable program.

  • Preprocessing
  • Compilation
  • Assembly
  • Linking
  • Execution
