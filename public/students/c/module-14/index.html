Module 14 — Preprocessor & Macros

Preprocessor & Macros in C

The C preprocessor processes code before compilation. Learn macros, header files, conditional compilation, and user-defined directives.

1. What is a Preprocessor?

A tool that modifies code before actual compilation.

  • Handles macros
  • Includes header files
  • Performs conditional compilation
  • Textual substitutions
#include <stdio.h>
#define PI 3.14

2. #define (Object-like Macros)

#define MAX 100
#define PI 3.14159

int arr[MAX];
printf("%f", PI);

3. Function-like Macros

Behave like inline functions.

#define square(x)  ((x)*(x))

printf("%d", square(5)); // 25

4. #include — Header Files

Used to include libraries and user-defined headers.

  • #include <stdio.h> → system header
  • #include "myfile.h" → user header in same directory
// myfile.h
void greet();

// main.c
#include "myfile.h"

5. Conditional Compilation

Compile specific parts of code based on conditions.

#define DEBUG 1

#if DEBUG
    printf("Debug mode ON");
#endif
    
#ifdef WINDOWS
    // windows-specific code
#else
    // linux code
#endif

6. #undef

Used to remove a macro definition.

#define LIMIT 50
#undef LIMIT

7. Predefined Macros

printf("%s", __FILE__);   // file name
printf("%d", __LINE__);   // current line number
printf("%s", __DATE__);   // compile date
printf("%s", __TIME__);   // compile time

8. Creating User-Defined Header Files

// mathutils.h
int add(int a, int b){
    return a + b;
}

// main.c
#include "mathutils.h"
printf("%d", add(3, 7));
