Operators & Expressions

Learn how C performs calculations, comparisons, logical decisions, and bitwise operations using its powerful operator system.

1. Arithmetic Operators

Used for mathematical operations.

  • + Addition
  • - Subtraction
  • * Multiplication
  • / Division
  • % Modulus (remainder)
int a = 10, b = 3;
printf("%d", a + b); // 13
printf("%d", a % b); // 1

2. Relational Operators

Used for comparison; result is TRUE(1) or FALSE(0).

  • == Equal to
  • != Not equal to
  • > Greater than
  • < Less than
  • >= Greater or equal
  • <= Less or equal
int x = 5, y = 10;
printf("%d", x < y);   // 1 (true)

3. Logical Operators

Used to combine multiple conditions.

  • && Logical AND
  • || Logical OR
  • ! Logical NOT
int age = 20;
printf("%d", age > 18 && age < 30);  // 1 (true)

4. Increment & Decrement

Increase or decrease variable values.

  • ++a Prefix increment
  • a++ Postfix increment
  • --a Prefix decrement
  • a-- Postfix decrement
int a = 5;
printf("%d", ++a); // 6 (prefix)
printf("%d", a++); // 6 (then becomes 7)

5. Bitwise Operators

Operate at the binary level.

  • & AND
  • | OR
  • ^ XOR
  • << Left shift
  • >> Right shift
int a = 5, b = 3;
printf("%d", a & b); // 1
printf("%d", a << 1); // 10

6. Operator Precedence

Defines which operators execute first.

  • Highest: (), ++, --
  • Medium: *, /, %
  • Lower: +, -
  • Lowest: =
int x = 10 + 20 * 3;  // 10 + 60 = 70
