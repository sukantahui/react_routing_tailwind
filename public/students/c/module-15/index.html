Module 15 — Advanced C Programming

Advanced C Programming

Explore advanced C concepts including enums, typedef, function pointers, command-line arguments, bit-fields, and modular programming.

1. Command Line Arguments

Pass values to a program during execution.

int main(int argc, char *argv[]) {
    printf("Arguments count: %d\n", argc);
    printf("First argument: %s\n", argv[1]);
}
  • argc → number of arguments
  • argv → array of string arguments

2. Enumerations (enum)

User-defined type for naming constant integer values.

enum Day { MON, TUE, WED, THU, FRI, SAT, SUN };

enum Day today = WED;
printf("%d", today); // 2

3. typedef

Creates aliases for existing types.

typedef unsigned long UL;

UL population = 7800000000;
    
// struct alias
typedef struct {
    int roll;
    char name[20];
} Student;

Student s1;

4. Bit-fields

Store data efficiently in limited memory.

struct flags {
    unsigned int bold : 1;
    unsigned int italic : 1;
    unsigned int underline : 1;
};

struct flags f = {1, 0, 1};
  • Used in embedded systems
  • Memory-efficient representation

5. Function Pointers

Store addresses of functions and call them dynamically.

void greet() {
    printf("Hello Students!");
}

void (*fp)() = greet;

fp();  // calls greet()
    
// Function pointer with parameters
int add(int a, int b){ return a + b; }

int (*ptr)(int,int) = add;

printf("%d", ptr(3,4));  // 7

6. Callback Functions

Passing a function as an argument to another function.

void process(int (*fn)(int, int)) {
    printf("%d", fn(5, 3));
}

int multiply(int a, int b) { return a * b; }

process(multiply);  // callback

7. Modular Programming

Break large projects into multiple files.

// mathutils.c
int add(int a, int b){ return a + b; }

// mathutils.h
int add(int a, int b);

// main.c
#include "mathutils.h"
printf("%d", add(4, 6));
  • Better maintainability
  • Team-friendly
  • Enhances readability

8. Memory Layout of a C Program

  • Text Segment — code
  • Data Segment — global & static variables
  • Heap — dynamic memory
  • Stack — function calls & local variables
int globalVar;      // data segment
static int s = 10;  // data segment

void func() {
    int x = 5;      // stack
    int *p = malloc(sizeof(int)); // heap
}
