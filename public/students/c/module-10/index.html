Dynamic Memory Allocation in C

C allows allocating memory at runtime using malloc, calloc, realloc, and free. Essential for building flexible, efficient programs and data structures.

1. What is Dynamic Memory?

Memory allocated during runtime (not compile time).

  • Stored in the Heap
  • Flexible and scalable
  • Must be freed manually
  • Used for dynamic arrays, linked lists, trees, graphs, etc.
// Static
int a[10];

// Dynamic
int *p = (int*) malloc(10 * sizeof(int));

2. malloc()

Allocates memory but does NOT initialize it.

int *p = (int*) malloc(5 * sizeof(int));

if(p == NULL){
    printf("Memory not allocated");
}

3. calloc()

Allocates + initializes memory to ZERO.

int *p = (int*) calloc(5, sizeof(int));

4. realloc()

Resizes previously allocated memory.

int *p = (int*) malloc(3 * sizeof(int));

p = (int*) realloc(p, 6 * sizeof(int));  // resize

5. free()

Releases allocated memory back to system.

free(p);   // Must be called
