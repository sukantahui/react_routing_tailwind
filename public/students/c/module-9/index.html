Module 9 — Pointers

Pointers in C

Pointers store memory addresses instead of values. Understanding pointers gives you complete control over memory, arrays, functions, and dynamic allocation.

1. What is a Pointer?

A pointer is a variable that stores the address of another variable.

  • Uses * for declaration
  • Uses & (address-of) to fetch address
  • Used for memory manipulation
int a = 10;
int *p;

p = &a;    // p stores address of a

printf("%d", *p);  // 10 (dereferencing)

2. Address-of (&) and Dereference (*) Operators

Two operators that make pointers powerful.

int x = 50;

printf("%p", &x); // address
printf("%d", *(&x)); // value at that address (50)

3. Pointer Declaration & Initialization

int a = 5;
int *p = &a;
    
// Pointer to float
float f = 3.14;
float *pf = &f;

4. Pointer Arithmetic

Incrementing/decrementing pointers moves them by data-size.

int a = 10;
int *p = &a;

p++;   // moves ahead by 4 bytes on 32-bit/64-bit systems
    
// Works with arrays
int arr[3] = {10, 20, 30};
int *ptr = arr;

printf("%d", *(ptr + 1));  // 20

5. Pointers & Arrays

Array name behaves like a pointer to its first element.

int arr[3] = {5, 10, 15};

printf("%d", arr[0]);
printf("%d", *arr);       // same as arr[0]
printf("%d", *(arr + 2)); // 15

6. Pointer to Pointer (Double Pointer)

int a = 10;
int *p = &a;
int **pp = &p;

printf("%d", **pp); // 10

7. Pointers in Functions

Used for call by reference.

void update(int *x){
    *x = 999;
}

int main(){
    int a = 10;
    update(&a);
    printf("%d", a); // 999
}

8. Pointers & Strings

char s[] = "Pointer";
char *p = s;

while(*p){
    printf("%c ", *p);
    p++;
}

9. Common Pointer Errors

  • Uninitialized pointer
  • Dangling pointer
  • Pointer to freed memory
  • Accessing out-of-bound memory
  • Incorrect pointer arithmetic
// Uninitialized pointer (danger!)
int *p;
*p = 10;   // ❌ runtime error
