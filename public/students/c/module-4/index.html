Module 4 — Decision Making

Control Statements in C (Decision Making)

Learn how C makes decisions using if, else-if, switch-case, and decision-control blocks. Mastering these statements helps you write logic-driven programs.

1. The if Statement

Executes a block when a condition is true.

  • Used for simple decision-making.
  • Condition must be TRUE (non-zero) to execute the block.
int age = 20;
if (age >= 18) {
    printf("You are eligible to vote.");
}

2. The if-else Statement

Runs one block if true, another if false.

int marks = 45;

if (marks >= 50) {
    printf("Pass");
} else {
    printf("Fail");
}

3. Nested if-else

An if inside another if.

  • Used when multiple decisions must be taken sequentially.
  • Be careful with indentation for readability.
int num = 5;

if (num > 0) {
    if (num % 2 == 0)
        printf("Positive & Even");
    else
        printf("Positive & Odd");
} else {
    printf("Not Positive");
}

4. Ladder if-else-if

Used when multiple conditions must be checked one by one.

int marks = 82;

if (marks >= 90) {
    printf("Grade A");
}
else if (marks >= 75) {
    printf("Grade B");
}
else if (marks >= 50) {
    printf("Grade C");
}
else {
    printf("Fail");
}

5. The switch Statement

Used when a variable is compared to multiple constant values.

  • Cleaner alternative to multiple else-if conditions.
  • Uses case labels.
  • break prevents fall-through.
int choice = 2;

switch(choice) {
    case 1: printf("Start"); break;
    case 2: printf("Stop"); break;
    case 3: printf("Pause"); break;
    default: printf("Invalid choice");
}

6. Using break & continue

Useful inside loops and switch statements.

  • break: Exits a loop or switch.
  • continue: Skips current iteration of loop.
// continue example
for(int i=1; i<=5; i++){
    if(i == 3) continue;
    printf("%d ", i); // Output: 1 2 4 5
}
