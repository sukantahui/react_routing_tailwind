Structures & Unions in C

Structures allow grouping variables of different types under one name, while unions allow storing different types in the same memory space.

1. What is a Structure?

A structure groups related variables of different data types.

  • Used for complex data (student, employee, product, etc.)
  • Each member has unique memory
  • Supports arrays, pointers, nested structures
struct Student {
    int roll;
    char name[20];
    float marks;
};

2. Declaring Structure Variables

struct Student s1;
struct Student s2 = {1, "Riya", 88.5};

3. Accessing Structure Members

s1.roll = 10;
strcpy(s1.name, "Sukanta");
s1.marks = 92.5;

printf("%d %s %f", s1.roll, s1.name, s1.marks);

4. Array of Structures

Useful for managing lists (students, employees, items).

struct Student arr[3];

for(int i=0;i<3;i++){
    scanf("%d %s %f", &arr[i].roll, arr[i].name, &arr[i].marks);
}

5. Nested Structures

struct Address {
    char city[20];
    int pin;
};

struct Student {
    int roll;
    struct Address adr;
};

6. Pointer to Structure

struct Student s = {1, "Rony", 88};

struct Student *p = &s;

printf("%d", p->roll);    // using -> operator

7. What is a Union?

A union stores different data types in the SAME memory location.

  • Memory shared by all members
  • Only one member stores valid data at a time
  • Useful for memory-efficient designs
union Data {
    int i;
    float f;
    char str[20];
};

8. Union Usage Example

union Data d;

d.i = 10;
printf("%d\n", d.i);

d.f = 3.14;      // overwrites integer
printf("%f\n", d.f);

9. Structure vs Union

  • Structure: each member has separate memory
  • Union: all members share the same memory
  • Struct size = sum of members
  • Union size = largest member
// Size demo
struct S { int a; float b; };  // size ≈ 8
union U { int a; float b; };   // size = 4
