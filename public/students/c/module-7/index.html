Module 7 — Strings

Strings in C

Strings in C are character arrays ending with a null character '\0'. Learn creation, input/output, manipulation, and standard library functions.

1. What is a String?

String = Array of characters ending with '\0'.

  • Strings use char arrays.
  • Always end with a NULL character.
  • C does not have a built-in string type.
// Example
char name[20] = "Sukanta";
printf("%s", name);

2. Declaring & Initializing Strings

Multiple ways to create strings.

// Method 1
char s1[] = "Hello";

// Method 2
char s2[10] = "World";

// Method 3 (character-by-character)
char s3[] = {'C','o','d','e','\0'};

3. String Input & Output

Using scanf, gets, puts.

char name[30];

printf("Enter name: ");
scanf("%s", name);   // stops at space

printf("Your name: %s", name);
    
// gets() takes full line (deprecated)
gets(name);

// puts() prints a string
puts(name);

4. Common String Functions

From string.h library.

  • strlen(s) – length of string
  • strcpy(dest, src) – copy string
  • strcat(dest, src) – concatenate
  • strcmp(s1, s2) – compare strings
char a[20] = "Hello";
char b[20] = "World";

printf("%ld", strlen(a));        // 5
strcat(a, b);                    // HelloWorld
strcpy(b, "Coder");              // b = Coder
printf("%d", strcmp(a, b));      // comparison

5. Manual String Traversal

Loop through each character.

char s[] = "AccoTax";

for(int i = 0; s[i] != '\0'; i++) {
    printf("%c ", s[i]);
}

6. String Manipulation Examples

Logic building with strings.

// Count vowels in a string
char s[] = "education";
int count = 0;

for(int i = 0; s[i] != '\0'; i++){
    if(s[i]=='a'||s[i]=='e'||s[i]=='i'||s[i]=='o'||s[i]=='u')
        count++;
}

printf("Vowels = %d", count);

7. Strings & Pointers

Pointer notation works naturally with strings.

char s[] = "Pointer";
char *p = s;

while(*p != '\0'){
    printf("%c ", *p);
    p++;
}
