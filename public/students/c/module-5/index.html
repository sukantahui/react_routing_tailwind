Looping Statements in C

Loops allow repeating a block of code multiple times. Master for, while, do-while, and nested loops to build efficient programs.

1. The for Loop

Used when the number of iterations is known.

  • Most commonly used loop in C.
  • Contains initialization, condition, increment/decrement in one line.
for(int i = 1; i <= 5; i++) {
    printf("%d ", i);
}
// Output: 1 2 3 4 5

2. The while Loop

Used when the number of iterations is not known.

int i = 1;

while(i <= 5) {
    printf("%d ", i);
    i++;
}

3. The do-while Loop

Executes at least once, even if condition is false.

int i = 1;

do {
    printf("%d ", i);
    i++;
} while(i <= 5);

4. Infinite Loops

A loop that never ends unless manually exited.

while(1) {
    printf("Running...");
}

5. Nested Loops

A loop inside another loop.

for(int i = 1; i <= 3; i++) {
    for(int j = 1; j <= 3; j++) {
        printf("%d ", j);
    }
    printf("\n");
}

6. Loop Control Statements

Modify the normal flow inside loops.

  • break: exits a loop immediately.
  • continue: skips current iteration.
// break example
for(int i = 1; i <= 5; i++) {
    if(i == 3) break;
    printf("%d ", i);
}

// continue example
for(int i = 1; i <= 5; i++) {
    if(i == 3) continue;
    printf("%d ", i);
}

7. Simple Menu Driven Program

Combines loops + switch.

int choice;

do {
    printf("1. Add\n2. Subtract\n3. Exit\n");
    scanf("%d", &choice);

    switch(choice) {
        case 1: printf("Addition\n"); break;
        case 2: printf("Subtraction\n"); break;
        case 3: printf("Exiting...\n"); break;
        default: printf("Invalid choice\n");
    }
} while(choice != 3);
