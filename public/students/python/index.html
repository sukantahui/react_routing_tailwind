Unit 1 Introduction to Programming & Python Basics

Understand problem-solving steps, algorithm design, and how Python fits as a high-level programming language.

Fundamental Concepts
1.1 Basics of Programming
  • Definition of program and programming.
  • Concept of algorithm and flowchart.
  • Problem-solving steps: analysis, design, coding, testing.
  • Structured vs. object-oriented programming (basic idea).
1.2 Introduction to Python
  • What is Python? Features & real-life applications.
  • Python 3.x installation (IDLE, VS Code / PyCharm).
  • Interactive mode vs. script mode.
  • Concept of indentation and its importance.
  • Running simple programs: print("Hello, Python").
1.3 Variables & Identifiers
  • Variables and memory concept (conceptual level).
  • Valid identifiers and naming conventions.
  • Reserved keywords in Python.
  • Single-line & multi-line comments.
1.4 Input / Output Basics
  • Using input() for user input.
  • Basic formatted output using print().
  • Simple examples combining input, processing, and output.
Unit 2 Data Types, Operators & Expressions

Learn how Python stores values, how expressions are formed, and how results are computed.

Core Python
2.1 Data Types
  • Numeric: int, float, complex.
  • bool – True/False.
  • str – strings and basic operations.
  • Collection types: list, tuple, set, dict (introduction).
  • Type checking & conversion: type(), int(), float(), str().
2.2 Operators
  • Arithmetic: + - * / // % **.
  • Relational / comparison: < > <= >= == !=.
  • Logical: and, or, not.
  • Assignment operators & compound assignments.
  • Membership: in, not in and identity: is, is not.
Exercise
2.3 Expressions
  • Forming arithmetic expressions.
  • Operator precedence & associativity.
  • Use of parentheses to control evaluation.
  • Simple expression-based problems (GST, discounts, averages).
Unit 3 Control Flow Statements

Control the flow of execution using decisions and loops to solve real-world problems.

Decision & Repetition
3.1 Conditional Statements
  • if statement.
  • if–else and nested if.
  • if–elif–else ladder.
  • Examples: grading system, max of three numbers, commission calculation.
Exercise
3.2 Looping Statements
  • while loop.
  • for loop with range().
  • Looping over strings, lists, and ranges.
  • Nested loops (basic patterns and tables).
Exercise
3.3 Loop Control
  • break – immediate exit from loop.
  • continue – skip current iteration.
  • pass as a placeholder.
  • Simple menu-driven programs using loops.
Exercise
Unit 4 String & List Processing

Process and analyze text data and work with lists for collections of values.

Data Handling
4.1 String Operations
  • Creating and indexing strings.
  • Slicing: s[start:end:step].
  • Common methods: upper(), lower(), title(), strip().
  • Searching and replacing: find(), replace().
  • Splitting and joining: split(), join().
  • Basic string formatting using format() and f-strings.
Exercise
4.2 Lists
  • Creating lists and accessing elements.
  • List slicing and modification.
  • List methods: append(), insert(), remove(), pop().
  • Sorting and reversing lists.
  • Nested lists and simple 2D list examples (e.g., marks of students).
  • Basic list comprehensions (optional, exam-oriented exposure).
Exercise
Unit 5 Functions in Python

Group reusable logic into functions to keep code structured and maintainable.

Reusability
5.1 Defining & Calling Functions
  • Syntax of def and function header.
  • Positional and keyword arguments.
  • Default arguments.
  • Variable-length arguments: *args and **kwargs (basic idea).
Exercise   Answer
5.2 Return Values
  • return statement – returning one value.
  • Returning multiple values as tuples.
  • Void functions vs value-returning functions.
Exercise Answer
5.3 Scope of Variables
  • Local vs global variables.
  • The global keyword (basic exposure).
  • Good practices for function-based programming.
Exercise Answer
Unit 6 Modules & Packages

Organize larger programs into multiple files and reuse standard Python libraries.

Standard Library
6.1 Modules
  • Importing modules using import and from ... import ....
  • Common built-in modules:
    • math – basic mathematical functions.
    • random – random numbers, simple simulations.
    • datetime – date and time basics.
  • Creating small user-defined modules.
06-01-Python-Modules
6.2 Packages (Intro)
  • Concept of package (folder with __init__.py).
  • Simple demonstration of importing from a package.
  • Using pip to install a third-party package (optional, conceptual).
Unit 7 File Handling in Python

Store and retrieve information using text files – essential for reports, logs, and persistent data.

Persistent Data
7.1 Text File Operations
  • Opening files with open().
  • File modes: "r", "w", "a", "r+" (basic).
  • Reading files: read(), readline(), readlines().
  • Writing & appending data using write() and writelines().
  • Closing files and using with context manager (intro).
7.2 Simple File Programs
  • Counting characters, words, and lines in a file.
  • Copying content from one file to another.
  • Searching for a word in a text file (simple search).
  • Maintaining a small text-based record (e.g., names and marks).
Unit 8 Data Structures: Tuples, Dictionaries & Sets

Learn Python’s built-in container types that help organize structured information.

Structured Data
8.1 Tuples
  • Definition and creation of tuples.
  • Tuple indexing and slicing.
  • Immutability concept.
  • Common operations & basic methods.
8.2 Dictionaries
  • Key-value pairs, creating dictionaries.
  • Accessing, adding, updating and deleting entries.
  • Useful methods: keys(), values(), items(), get().
  • Building simple look-up tables (e.g., code → description).
8.3 Sets
  • Creating sets, uniqueness of elements.
  • Adding and removing elements.
  • Set operations: union, intersection, difference.
  • Simple applications like duplicate removal.
Unit 9 Basics of Object-Oriented Programming in Python

Introductory view of classes and objects to connect with higher-level programming concepts.

Conceptual
9.1 Classes & Objects
  • Concept of class and object.
  • Defining a simple class with attributes and methods.
  • Creating objects and accessing members.
  • Use of __init__() as a constructor (simple examples).
9.2 Simple Inheritance (Intro)
  • Basic idea of derived class and base class (concept only).
  • Very simple example showing code reuse via inheritance.
  • OOP as a foundation for higher-level courses (Python/Java).
Unit 10 Searching & Sorting Algorithms

Apply Python to implement basic algorithms used in everyday computing tasks.

Algorithmic Thinking
10.1 Searching
  • Linear search on lists.
  • Binary search on sorted lists.
  • Analysis: when linear is sufficient vs when binary is efficient (intuitive understanding).
10.2 Sorting
  • Bubble sort (concept and implementation in Python).
  • Selection sort.
  • Insertion sort.
  • Using Python’s built-in sort() and sorted() for practical purposes.
Unit 11 Optional: Simple Data Visualization

Recommended enrichment topic for labs and project work using a library such as matplotlib.

Enrichment
11.1 Basic Charts
  • Installing and importing matplotlib (if allowed in environment).
  • Creating simple line charts from lists.
  • Bar charts for marks or sales data.
  • Pie charts for category-wise representation.
11.2 Lab Integration
  • Using charts to visualize search/sort performance (basic idea).
  • Turning tabular data into simple plots.
Lab Component Practicals & Mini Project (Suggested)

Hands-on Practice

  • Programs using:

    • List and string processing (search, count, reverse).
  • Number system & calculation based problems (simple).
  • Menu-driven programs for small tasks.
  • File reading and writing tasks.
