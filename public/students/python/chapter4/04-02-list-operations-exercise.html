1. Multiple Choice Questions (MCQs)

  1. Which of the following creates a list?
    a) (1, 2, 3) b) [1, 2, 3] c) {"a": 1} d) "123"
  2. Lists in Python are:
    a) Immutable b) Mutable c) Hybrid d) None
  3. append() adds element:
    a) At beginning b) At end c) At index 0 d) At index 1
  4. fruits.insert(1, "Apple") means:
    a) Replace at index 1 b) Insert at index 1 c) Delete index 1 d) Error
  5. Which removes element by value?
    a) pop() b) del c) remove() d) clear()
  6. [1,2,3] * 2 is:
    a) [1,2,3,1,2,3] b) [2,4,6] c) [1,2,3,2] d) None
  7. Which returns number of occurrences?
    a) index() b) count() c) sum() d) find()
