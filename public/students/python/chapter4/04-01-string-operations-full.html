1. Introduction to Strings

A string in Python is a sequence of characters enclosed within quotes. Strings are used to store text: names, messages, file paths, etc.

  • Strings are immutable (cannot be changed in-place).
  • Strings are sequence types ‚Äî we can index, slice and iterate over them.
Examples of strings: "Hello", 'Python', "123", "Coder & AccoTax"

2. Creating Strings

2.1 Using Single and Double Quotes

s1 = "Hello Python" s2 = 'Welcome to NIELIT O Level' print(s1) print(s2)

2.2 Triple Quotes for Multi-line Strings

Use ''' ... ''' or """ ... """ for multi-line strings.

msg = """This is a multi-line string in Python.""" print(msg)

2.3 Escape Characters

Some special characters:

  • \n ‚Üí newline
  • \t ‚Üí tab
  • \\ ‚Üí backslash
  • \' ‚Üí single quote
  • \" ‚Üí double quote
text = "Hello\nWorld\t2025" print(text)

3. Indexing & Slicing Strings

Each character in a string has a position called an index.

  • Indexing starts from 0 (left to right).
  • Negative indices start from -1 (right to left).

3.1 Indexing Diagram

text = "PYTHON" Index ( + ): P Y T H O N 0 1 2 3 4 5 Index ( - ): P Y T H O N -6 -5 -4 -3 -2 -1

3.2 Accessing Characters

text = "PYTHON" print(text[0]) # P print(text[3]) # H print(text[-1]) # N print(text[-3]) # H

3.3 Slicing Strings

Slice syntax:

substring = s[start : stop : step]
‚Ä¢ start ‚Üí starting index (included)
‚Ä¢ stop ‚Üí ending index (excluded)
‚Ä¢ step ‚Üí jump (default 1)

Examples

s = "PYTHON PROGRAM" print(s[0:6]) # 'PYTHON' print(s[7:14]) # 'PROGRAM' print(s[:6]) # from start ‚Üí 'PYTHON' print(s[7:]) # from index 7 to end ‚Üí 'PROGRAM' print(s[-7:-1]) # 'PROGRA' print(s[::-1]) # reversed string

4. Basic String Operations

  • Concatenation ‚Üí +
  • Repetition ‚Üí *
  • Membership test ‚Üí in, not in
  • Length ‚Üí len()
a = "Hello" b = "Python" print(a + " " + b) # concatenation print(a * 3) # HelloHelloHello print("Py" in b) # True print(len(b)) # 6

5. Important String Methods

5.1 Changing Case ‚Äî upper(), lower(), title(), capitalize()

MethodDescription
upper()Converts all characters to uppercase
lower()Converts all characters to lowercase
title()First letter of each word in uppercase
capitalize()First character of string in uppercase, rest lowercase
name = "coder & accotax python course" print(name.upper()) print(name.lower()) print(name.title()) print(name.capitalize())

5.2 Removing Spaces ‚Äî strip(), lstrip(), rstrip()

MethodDescription
strip()Removes spaces from both ends
lstrip()Removes spaces from left side
rstrip()Removes spaces from right side
data = " Python Programming " print("->" + data + "<-") print("->" + data.strip() + "<-") print("->" + data.lstrip() + "<-") print("->" + data.rstrip() + "<-")

5.3 Searching in Strings ‚Äî find(), rfind()

MethodDescription
find(sub)Returns first index of substring, or -1 if not found
rfind(sub)Returns last index of substring
text = "welcome to python programming" print(text.find("python")) # index where 'python' starts print(text.find("Java")) # -1 (not found) print(text.rfind("o")) # last index of 'o'

5.4 Replacing Text ‚Äî replace()

replace(old, new) returns a new string with replacements done.

msg = "I love Java. Java is powerful." new_msg = msg.replace("Java", "Python") print(new_msg)

5.5 Checking Start/End ‚Äî startswith(), endswith()

site = "www.codernaccotax.co.in" print(site.startswith("www")) # True print(site.endswith(".in")) # True print(site.endswith(".com")) # False

5.6 Counting Occurrences ‚Äî count()

msg = "banana" print(msg.count("a")) # 3 print(msg.count("an")) # 2

6. Splitting and Joining Strings

6.1 split()

split() breaks a string into a list of substrings based on a separator (default = space).

line = "Python is fun to learn" words = line.split() # split by space print(words) # ['Python', 'is', 'fun', 'to', 'learn'] date = "2025-11-23" parts = date.split("-") print(parts) # ['2025', '11', '23']

6.2 join()

join() is used on a separator string to join list elements into a string.

words = ["Coder", "&", "AccoTax"] sentence = " ".join(words) print(sentence) # Coder & AccoTax date_parts = ["2025", "11", "23"] date_str = "-".join(date_parts) print(date_str) # 2025-11-23
Remember: "separator".join(list) is the correct pattern. The list elements must all be strings.

7. String Formatting ‚Äî format() and f-strings

String formatting helps you create readable strings by injecting values into templates.

7.1 Using format()

Positional placeholders

name = "Sukanta" course = "Python" score = 95 msg = "Student {0} scored {1} in {2}".format(name, score, course) print(msg)

Named placeholders

msg = "Welcome {user}, your balance is ‚Çπ{bal}".format(user="Rahul", bal=1500) print(msg)

7.2 Using f-strings (Python 3.6+)

f-strings are the modern and most convenient way to format strings.

name = "Sukanta" course = "Python" score = 95 msg = f"Student {name} scored {score} in {course}" print(msg)

Expressions inside f-strings

a = 5 b = 3 print(f"{a} + {b} = {a + b}") print(f"Square of {a} is {a**2}")

Formatting numbers

pi = 3.1415926535 print(f"Pi rounded to 2 decimals: {pi:.2f}") amount = 123456.789 print(f"Amount = ‚Çπ{amount:,.2f}")
