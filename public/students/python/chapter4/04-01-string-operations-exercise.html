1. Multiple Choice Questions (MCQs)

  1. Which indexing starts from the right side?
    a) Reverse indexing b) Negative indexing c) Back indexing d) None
  2. len("Python") gives:
    a) 5 b) 6 c) 7 d) 0
  3. Which method splits a string?
    a) join() b) upper() c) split() d) replace()
  4. "hello".upper() returns:
    a) Hello b) HELLO c) hello d) Error
  5. "apple".find("p") returns:
    a) 0 b) 1 c) 2 d) None
  6. "banana".count("an") returns:
    a) 1 b) 2 c) 3 d) 4
  7. Which of the following creates a multi-line string?
    a) '' b) "" c) '''...''' d) All
  8. "-".join(["a", "b", "c"]) returns:
    a) a b c b) a-b-c c) abc d) None
  9. Which is mutable?
    a) list b) string c) both d) none
  10. Which removes spaces from both ends?
    a) strip() b) split() c) trim() d) clean()

3. Slicing Practice

Given:

s = "CODER & ACCOTAX" Indexes (+): 0 1 2 3 4 5 ... C O D E R & Indexes (-): -15 -14 ... -1
  1. Print "CODER"
  2. Print "ACCOTAX"
  3. Reverse the entire string
  4. Print characters from index 3 to 8
  5. Print every 2nd character
  6. Print last 5 characters using negative slicing
  7. Print "& ACCO" using slice
  8. Print "CR&AO" using steps
  9. Print first 5 characters in reverse order
  10. Print all characters except first and last

4. Coding Tasks

Easy

  • Input a name and print it in upper(), lower(), title().

  • Remove all spaces from a string.

Moderate
