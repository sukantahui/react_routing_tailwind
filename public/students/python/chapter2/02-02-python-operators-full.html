1. Introduction to Operators

An operator is a special symbol that tells Python to perform a specific operation (like addition, comparison, logical test) on one or more operands.

Example:

x = 10 y = 3 z = x + y

Here:

  • x and y are operands.
  • + is the operator.
  • z stores the result of the expression x + y.
Expression: Any combination of values, variables and operators that Python can evaluate and produce a result. Example: (a + b) * 2.

2. Arithmetic Operators

Used for mathematical operations on numeric data types.

Operator Meaning Example Result
+ Addition 10 + 5 15
- Subtraction 10 - 5 5
* Multiplication 10 * 5 50
/ Division (float) 10 / 4 2.5
// Floor Division 10 // 4 2
% Modulus (remainder) 10 % 4 2
** Exponent (power) 2 ** 3 8
Example ‚Äî Basic arithmetic
a = 15 b = 4 print("a + b =", a + b) print("a - b =", a - b) print("a * b =", a * b) print("a / b =", a / b) print("a // b =", a // b) print("a % b =", a % b) print("a ** b =", a ** b)

2.1 Difference Between / and //

print(10 / 3) # 3.3333333333 (float) print(10 // 3) # 3 (integer part only) print(-10 / 3) # -3.3333333333 print(-10 // 3) # -4 (floor division moves towards -infinity)
Exam Tip: Many objective questions ask the result of expressions using // and %, especially with negative numbers.

3. Relational (Comparison) Operators

These operators compare two values and return a boolean result: True or False.

Operator Meaning Example Result
== Equal to 5 == 5 True
!= Not equal to 5 != 3 True
> Greater than 7 > 2 True
< Less than 7 < 2 False
>= Greater than or equal 5 >= 5 True
<= Less than or equal 3 <= 4 True
Example ‚Äî Pass / Fail check
marks = 68 is_pass = marks >= 40 print("Pass status:", is_pass) age = 17 print(age >= 18) # False

3.1 Comparing Strings

Strings are compared using dictionary (lexicographical) order based on Unicode values of characters.

print("apple" == "apple") # True print("apple" < "banana") # True (because 'a' < 'b') print("Zebra" < "apple") # True ('Z' has smaller code than 'a')

4.2 Short-Circuit Evaluation

In expressions like cond1 and cond2, if cond1 is False, Python does not evaluate cond2. Similarly, with or, if cond1 is True, it doesn‚Äôt evaluate cond2.
def check(): print("check() called") return True print(False and check()) # check() is NOT called print(True or check()) # check() is NOT called

Used to assign values to variables. Combined assignment operators perform an operation and assignment together.

6. Bitwise Operators

Bitwise operators work on the binary representation of integers. Useful in low-level programming, flags, permissions, etc.

Operator Meaning Example
& Bitwise AND a & b
| Bitwise OR a | b
^ Bitwise XOR (exclusive OR) a ^ b
~ Bitwise NOT ~a
<< Left shift a << 1
>> Right shift a >> 1
Example ‚Äî Bitwise basics
a = 5 # binary: 0101 b = 3 # binary: 0011 print(a & b) # 1 -> 0001 print(a | b) # 7 -> 0111 print(a ^ b) # 6 -> 0110 print(a << 1) # 10 -> 1010 print(a >> 1) # 2 -> 0010
Bitwise NOT (~a): In Python, ~a is equal to -(a+1). Example: ~5 = -6.

7. Membership Operators

Used to test whether a value is present in a sequence (string, list, tuple, set, dictionary keys).

Operator Meaning Example
in True if value present "a" in "apple"
not in True if value not present "z" not in "apple"
fruits = ["apple", "banana", "mango"] print("apple" in fruits) # True print("orange" in fruits) # False print("orange" not in fruits) # True text = "Python Programming" print("thon" in text) # True

8. Identity Operators

Identity operators check whether two variables point to the same object in memory.

Operator Meaning Example
is True if both refer to same object x is y
is not True if they refer to different objects x is not y
Example ‚Äî is vs ==
a = [1, 2, 3] b = a c = [1, 2, 3] print(a == b) # True (values equal) print(a is b) # True (same object) print(a == c) # True (values equal) print(a is c) # False (different objects)
Remember: Use == to compare values, use is to compare identity (object reference).

9. Unary Operators

Operators that act on a single operand.

  • +x ‚Üí unary plus (usually no effect)
  • -x ‚Üí unary minus (changes sign)
  • ~x ‚Üí bitwise NOT (as seen earlier)
x = 5 print(+x) # 5 print(-x) # -5 y = -7 print(-y) # 7

10. Ternary (Conditional) Expression

Python does not have a traditional ?: operator like C, but uses a conditional expression:

result = value_if_true if condition else value_if_false
Example ‚Äî Larger of two numbers
a = 10 b = 20 max_value = a if a > b else b print("Maximum =", max_value)
Example ‚Äî Even or Odd
n = int(input("Enter a number: ")) result = "Even" if n % 2 == 0 else "Odd" print("Number is", result)

11. Chained Comparisons

Python allows chaining of comparison operators in a very natural way: 5 < x < 10 is allowed.

x = 7 print(5 < x < 10) # True print(5 < x and x < 10) # Same as above age = 18 print(13 <= age <= 19) # True (Teenager)

12. Operator Precedence & Associativity

When multiple operators appear in one expression, Python follows a priority order called operator precedence. If operators have the same precedence, associativity decides evaluation direction (left to right or right to left).

12.1 Precedence Table (Simplified)

Precedence Operators Associativity
Highest ( ) (Parentheses) ‚Äì
** Right to Left
+x, -x, ~x (Unary) Right to Left
*, /, //, % Left to Right
+, - Left to Right
Comparison: <, <=, >, >=, ==, != Left to Right
not Right to Left
and Left to Right
or Left to Right
Lowest =, +=, -=, ... Right to Left

12.2 Precedence Examples

expr1 = 5 + 3 * 2 # * has higher precedence than + # 3 * 2 = 6, then 5 + 6 = 11 print(expr1) # 11 expr2 = (5 + 3) * 2 # Parentheses first: 5 + 3 = 8, then 8 * 2 = 16 print(expr2) # 16 expr3 = 10 - 3 * 2 ** 2 # ** first: 2 ** 2 = 4 # then *: 3 * 4 = 12 # then -: 10 - 12 = -2 print(expr3) # -2
expr4 = 10 > 5 and 3 < 1 # 10 > 5 -> True # 3 < 1 -> False # True and False -> False print(expr4) # False
Best practice: Use parentheses ( ) to make expressions easier to read and to avoid confusion in complex conditions.
