1. Introduction to Data Types

In programming, a data type describes what kind of value a variable can hold and what operations can be performed on that value. For example, you can add two numbers, but you cannot ‚Äúadd‚Äù a number and a list without converting them properly.

Key idea: Data types give structure to data. Python automatically decides the data type when you assign a value, because it is a dynamically typed language.
Quick Example ‚Äî Different Types in One Program
roll = 101 # int name = "Anita" # str marks = 87.5 # float is_pass = True # bool print(type(roll)) print(type(name)) print(type(marks)) print(type(is_pass))

2. Numeric Data Types (Numbers)

Python supports three main numeric data types:

  • int ‚Äî integer (whole numbers)
  • float ‚Äî decimal / real numbers
  • complex ‚Äî complex numbers (a + bj form)

2.1 Integer (int)

Integers are whole numbers without decimal points. They can be positive, negative or zero. There is no fixed limit on the size of an integer in Python (only limited by memory).

Examples of int
a = 10 b = -250 c = 0 big = 98765432123456789 print(a, b, c, big) print(type(a)) # <class 'int'>

Common integer operations

  • Arithmetic: +, -, *, //, %, **
  • Use in loops, counters, indices
Mini Example ‚Äî Simple Billing
quantity = 5 price_per_item = 120 total = quantity * price_per_item print("Total amount =", total)

2.2 Float (float)

Floats are numbers with a decimal point or in exponent (scientific) notation. Used for measurements, marks with decimal, currency, etc.

Examples of float
pi = 3.14159 gst_rate = 18.0 distance = 12.5 big_float = 1.2e3 # 1.2 √ó 10¬≥ = 1200.0 print(pi, distance, big_float) print(type(pi)) # <class 'float'>
Note: Floating point values may not be 100% exact in binary representation, so sometimes calculations like 0.1 + 0.2 may give 0.30000000000000004. This is common in many languages.

2.3 Complex (complex)

Complex numbers are written in a + bj form, where a is the real part and b is the imaginary part.

Complex Number Usage
z1 = 2 + 3j z2 = 4 - 5j print("z1 =", z1) print("Real part of z1:", z1.real) print("Imag part of z1:", z1.imag) z3 = z1 + z2 print("z1 + z2 =", z3)

3. Boolean (bool)

The Boolean data type has only two values: True or False. It is mainly used with conditions and decision-making (if, while, etc.).

Booleans from Comparisons
x = 10 y = 20 print(x < y) # True print(x == y) # False print(x != y) # True is_valid = (x < y) and (x > 0) print(is_valid) # True
Truthiness: In Python, some values are treated as False in boolean context: 0, 0.0, "", [], (), {}, None. Everything else is considered True.

4. Strings (str)

Strings are sequences of characters used to store text ‚Äî names, address, messages, etc. You can use single quotes, double quotes or triple quotes.

4.1 Creating and Printing Strings

title = "Python Programming" author = 'Sukanta' multi_line = """This is a multi-line string.""" print(title) print(author) print(multi_line)

4.2 String Indexing and Slicing

Strings are indexed from 0. You can access individual characters or slices (substrings).

name = "CODER ACCOTAX" print(name[0]) # C print(name[1]) # O print(name[-1]) # X (last character) print(name[0:5]) # CODER print(name[6:]) # ACCOTAX print(name[:5]) # CODER

4.3 Common String Operations

Important methods:
s = " hello Python " print(s.upper()) # HELLO PYTHON print(s.lower()) # hello python print(s.strip()) # "hello Python" (removes side spaces) print(s.replace("Python", "World")) # " hello World " print("Length:", len(s)) # includes spaces
Exam Tip: You will often get questions to find output of string slicing or len(), so practice manually tracing indices.

5. Lists

A list is an ordered, mutable collection. You can:

  • Store multiple values in one variable
  • Allow duplicates
  • Change, add, or remove elements

5.1 Creating Lists

marks = [78, 85, 90, 66] mixed = [10, "Anita", True, 87.5] print(marks) print(mixed)

5.2 Accessing and Modifying Lists

fruits = ["apple", "banana", "mango", "orange"] print(fruits[0]) # apple print(fruits[-1]) # orange fruits[1] = "grapes" # modify print(fruits) # ['apple', 'grapes', 'mango', 'orange']

5.3 Common List Methods

nums = [5, 2, 9, 1] nums.append(7) # add at end nums.insert(1, 10) # insert 10 at index 1 nums.sort() # ascending sort nums.reverse() # reverse order print(nums)
When to use a list? When you need an ordered collection that you can modify frequently (add/remove/update elements).

6. Tuples

A tuple is like a list, but immutable ‚Äî once created, you cannot change it. Tuples are often used when you want to protect data from accidental modification.

6.1 Creating and Using Tuples

point = (10, 20, 30) days = ("Mon", "Tue", "Wed", "Thu", "Fri", "Sat", "Sun") print(point[0]) # 10 print(days[2]) # Wed # point[1] = 50 # ‚ùå Error: tuple does not support item assignment
Use Tuple: For fixed data like days of week, months, configurations that should not change.

7. Sets

A set is an unordered collection of unique elements. Useful for operations like union, intersection, removing duplicates.

7.1 Creating Sets

nums = {10, 20, 30, 20, 10} print(nums) # {10, 20, 30} ‚Üí duplicates removed empty_set = set() # correct way to create empty set

7.2 Set Operations

a = {1, 2, 3, 4} b = {3, 4, 5, 6} print("Union:", a | b) # {1,2,3,4,5,6} print("Intersection:", a & b) # {3,4} print("Difference:", a - b) # {1,2}
Important: Sets are unordered. You cannot access elements by index like s[0].

8. Dictionaries

A dictionary stores data as key‚Äìvalue pairs. Keys must be unique, values can repeat.

8.1 Creating and Accessing Dictionaries

student = { "name": "Rahul", "roll": 5, "course": "Python", "marks": 88 } print(student["name"]) print(student["marks"]) # add new key student["city"] = "Kolkata" print(student)

8.2 Dictionary Methods

print(student.keys()) # view all keys print(student.values()) # view all values print(student.items()) # view (key, value) pairs # safe access using get() print(student.get("name")) # Rahul print(student.get("phone")) # None (no error)
Use dictionary: when you want to store related information about an entity using labels (keys) like student with name, roll, marks, etc.

9. Type Conversion & Casting

Type conversion means converting a value from one data type to another. Python does some conversions automatically (implicit), but most of the time we use explicit conversion functions.

9.1 Common Casting Functions

  • int(x) ‚Äî convert to integer
  • float(x) ‚Äî convert to float
  • str(x) ‚Äî convert to string
  • list(x) ‚Äî convert to list
  • tuple(x) ‚Äî convert to tuple
  • set(x) ‚Äî convert to set
Numeric Casting
a = "25" b = "3.5" print(int(a)) # 25 print(float(a)) # 25.0 print(float(b)) # 3.5 x = 10.9 print(int(x)) # 10 (fraction removed)
Collection Casting
s = "ABC" print(list(s)) # ['A', 'B', 'C'] print(tuple(s)) # ('A', 'B', 'C') my_list = [1, 2, 2, 3] print(set(my_list)) # {1, 2, 3} (duplicates removed)
Be careful: If the string doesn‚Äôt look like a valid number, int() or float() will raise an error. Example: int("ABC") ‚Üí ValueError.

10. Taking User Input using input()

The built-in function input() allows you to take input from the keyboard during program execution. It always returns a string.

10.1 Basic Usage

name = input("Enter your name: ") print("Hello", name)
Important: input() ‚Üí always returns data as str. For numeric calculations, convert using int() or float().

10.2 Reading Numeric Input

Example ‚Äî Simple Interest
P = float(input("Enter Principal: ")) R = float(input("Enter Rate of Interest: ")) T = float(input("Enter Time (in years): ")) SI = (P * R * T) / 100 print("Simple Interest =", SI)

10.3 Reading Multiple Values

# Enter: 10 20 30 a, b, c = input("Enter three numbers separated by space: ").split() # a, b, c are strings here a = int(a) b = int(b) c = int(c) print("Sum =", a + b + c)
