1. Multiple Choice Questions (MCQs)

  1. Which keyword is used to define a function in Python?
    a) function   b) define   c) def   d) fn
  2. What will be the output?
    def f(a,b):
    return a+b
print(f(2,3))
    a) 23   b) Error   c) 5   d) None
  3. Arguments passed in the correct order without keywords are called:
    a) Keyword arguments   b) Positional arguments   c) Default arguments   d) Named arguments
  4. What is the type of parameter *args?
    a) list   b) tuple   c) tuple   d) string
  5. What is the type of parameter **kwargs?
    a) list   b) tuple   c) string   d) dictionary
  6. Which of the following is valid?
    def fun(a,b=10): pass
    a) Valid   b) Invalid (default must come first)   a) Valid
  7. What happens if you return nothing from a function?
    a) 0   b) Empty string   c) None   d) False
  8. Which function prints documentation of another function?
    a) doc()   b) print()   c) help()   d) info()
  9. Which keyword is used to return a value from a function?
    a) break   b) exit   c) return   d) stop
  10. Which of these will cause an error?
    def f(a,b,c): pass
f(a=10, 20, 30)
    a) Valid   b) Error
