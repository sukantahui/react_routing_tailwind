1. Introduction to Return Values

When a function completes its task, it can return a value back to the caller using the return statement. These return values allow functions to be used inside expressions, calculations, and data-processing tasks.

2. Returning a Single Value

Syntax:

def function_name():
    return value
Example 1 — Simple Sum
def add(a, b):
    return a + b

result = add(10, 5)
print(result)  # 15
Example 2 — Returning a String
def greet(name):
    return "Hello, " + name

print(greet("Sukanta"))

3. Returning Multiple Values (Tuples)

Python allows returning multiple values, which are automatically packed into a tuple.

Example 1 — Returning 3 values
def calculate(a, b):
    return a + b, a - b, a * b

result = calculate(10, 5)
print(result)   # (15, 5, 50)
Example 2 — Unpacking the Returned Tuple
sum_v, diff_v, prod_v = calculate(10, 5)
print(sum_v, diff_v, prod_v)

4. Void Functions vs Value-returning Functions

A void function performs an action but returns nothing. A value-returning function sends data back using return.

4.1 Void Function

def show_message():
    print("This is a void function")

print(show_message())
# Output:
# This is a void function
# None

4.2 Value-Returning Function

def square(n):
    return n * n

print(square(6))   # 36

5. Real-world Examples

Example — Function Chaining
def multiply(a, b):
    return a * b

def add_ten(x):
    return x + 10

print(add_ten(multiply(3, 4)))  # 22
Example — Returning Lists
def factors(n):
    return [i for i in range(1, n+1) if n % i == 0]

print(factors(12))
Example — Returning Dictionaries
def student_profile(name, age):
    return {"name": name, "age": age}

print(student_profile("Sudeshna", 16))

6. Common Mistakes with Return

  • Writing code after return (it becomes unreachable)
  • Returning inside a loop unintentionally
  • Using print() instead of return
def wrong():
    return "Done"
    print("This will never execute")
