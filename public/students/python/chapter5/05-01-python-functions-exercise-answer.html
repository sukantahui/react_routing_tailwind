Python Functions — Full Answer Key & Explanations

MCQs · Short Answers · Solved Coding Exercises (NIELIT O Level – 5.1)

1. MCQ Answers with Explanations

  1. Q1. Which keyword is used to define a function in Python?
    Answer: c) def
    Python uses the def keyword to define functions. Other options are not valid Python keywords for function definition.
  2. Q2. What will be the output of the following code?
    def f(a, b):
    return a + b

print(f(2, 3))
    Answer: c) 5
    The function f returns the sum of a and b. So 2 + 3 = 5, which is printed.
  3. Q3. Arguments passed in the correct order without using names are called:
    Answer: b) Positional arguments
    Positional arguments are matched by their position in the function call. Keyword arguments are matched by parameter name.
  4. Q4. What is the type of the parameter *args inside a function?
    Answer: c) tuple
    Python collects all extra positional arguments into a tuple when you use *args.
  5. Q5. What is the type of the parameter **kwargs inside a function?
    Answer: d) dictionary
    Python collects all extra keyword arguments into a dictionary when you use **kwargs, with keys as argument names.
  6. Q6. Consider the function definition:
    def fun(a, b=10):
    pass
    Is this valid in Python?
    Answer: Yes, it is valid
    Default arguments are allowed, but they must come after non-default parameters. Here a is non-default and b is default, so the order is correct.
  7. Q7. What is the value returned by a function that has no return statement?
    Answer: c) None
    If Python reaches the end of a function without executing a return, it automatically returns None.
  8. Q8. Which built-in function can be used to display the documentation (docstring) of another function?
    Answer: c) help()
    The help() function shows the documentation of modules, classes and functions, including their docstrings.
  9. Q9. Which keyword is used to send a value back from a function?
    Answer: c) return
    return ends the function execution and sends the specified value back to the caller.
  10. Q10. What happens with the following code?
    def f(a, b, c):
    pass

f(a=10, 20, 30)
    Answer: It causes an error
    In a function call, positional arguments must come before keyword arguments. Here, a=10 is a keyword argument followed by positional arguments 20, 30, so Python raises a SyntaxError.

2. Short Questions — Model Answers

  1. What is a function?
    A reusable block of code that performs a specific task.
  2. Why use functions?
    To reduce repetition, improve readability, and modularize code.
  3. Syntax of defining a function:
    def function_name(parameters):
    statements
  4. Difference between parameters and arguments?
    Parameters = in function definition; Arguments = in function call.
  5. What are positional arguments?
    Arguments passed in order; position matters.
  6. Keyword arguments:
    fun(name="Riya", age=20)
  7. Default arguments: Parameters with pre-assigned values.
  8. Why must default arguments come last?
    Otherwise Python cannot match values correctly.
  9. *args: Accepts variable-length positional arguments.
  10. **kwargs: Accepts variable-length keyword arguments.
  11. Return statement: Sends a result back from a function.
  12. Return multiple values?
    Yes, Python returns a tuple:
    return a, b
  13. No return?
    Function returns None.
  14. Variable-length arguments:
    *args and **kwargs allow flexibility.
  15. Advantage of keyword arguments:
    Order doesn’t matter; more readable.
  16. Function header:
    The first line starting with def.
  17. Function call:
    Executing a function by its name followed by ().
  18. Type of *args: Tuple.
  19. Difference *args vs **kwargs:
    *args → values; **kwargs → key-value.
  20. Parameter order:
    positional → default → *args → keyword-only → **kwargs

3. Solved Coding Exercises

1. Square of a number

Question: Write a function that returns the square of a number.

def square(n):
    return n * n

print(square(5))  # 25

2. Area of a circle

Question: Create a function area_circle(r) that returns the area of a circle.

def area_circle(r):
    return 3.1416 * r * r

print(area_circle(5))

3. Sum of list elements

Question: Write a function that takes a list and returns the sum of all its elements.

def sum_list(lst):
    return sum(lst)

print(sum_list([1, 2, 3]))

4. Product using *args

Question: Write a function using *args to calculate the product of all numbers passed to it.

def product(*nums):
    p = 1
    for n in nums:
        p *= n
    return p

print(product(2, 3, 4))

5. Print student details (**kwargs)

Question: Write a function using **kwargs to print student details in key: value format.

def student(**data):
    for k, v in data.items():
        print(k, ":", v)

student(name="Riya", age=20, course="O Level")

6. Sum & average

Question: Create a function that returns both the sum and average of 3 numbers.

def calc(a, b, c):
    s = a + b + c
    avg = s / 3
    return s, avg

print(calc(5, 10, 15))

def convert(c):

9. Count vowels

Question: Write a function that takes a string and returns the number of vowels.

def count_vowels(s):
    vowels = "aeiouAEIOU"
    count = 0
    for ch in s:
        if ch in vowels:
            count += 1
    return count

print(count_vowels("Python Programming"))

10. Largest of 3 numbers

Question: Write a function that returns the largest among 3 numbers.

def largest(a, b, c):

print(largest(10, 25, 5))

11. Simple calculator

Question: Write a calculator function that supports +, -, *, / using arguments.

def calc(a, b, op):
    if op == '+':
        return a + b
    elif op == '-':
        return a - b
    elif op == '*':
        return a * b
    elif op == '/':
        return a / b
    else:
        return "Invalid operator"

print(calc(10, 5, '*'))

def rev(s):

print(rev("hello"))

def words(s):

15. Prime check

Question: Write a function that checks whether a number is prime.

def is_prime(n):
    if n < 2:
        return False
    for i in range(2, int(n**0.5) + 1):
        if n % i == 0:
            return False
    return True

print(is_prime(17))

16. Default argument

Question: Write a function with a default argument country="India" that prints a welcome message.

def welcome(country="India"):
    print("Welcome to", country)

welcome()
welcome("USA")

17. Factorial

Question: Write a function that returns the factorial of a number.

def fact(n):
    f = 1
    for i in range(1, n + 1):
        f *= i
    return f

print(fact(6))

18. Remove duplicates from a list

Question: Write a function that removes duplicates from a list and returns the new list.

def remove_dup(lst):
    return list(set(lst))

print(remove_dup([1, 2, 2, 3, 3, 3]))

19. Print keys only (**kwargs)

Question: Write a function that accepts **kwargs and returns all keys.

def show_keys(**d):
    return list(d.keys())

print(show_keys(a=1, b=2, c=3))

20. Simple Interest

Question: Write a function to calculate simple interest.

def si(p, r, t):
    return (p * r * t) / 100

print(si(1000, 5, 2))
