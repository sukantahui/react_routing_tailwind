5.2 Return Values

Learn how functions return data using return, how multiple values are returned as tuples, and the difference between void & value-returning functions.

5.2.1 The return Statement

The return statement is used to send back a value from a function. Once return executes, the function stops immediately.

def add(a, b):
    return a + b

result = add(5, 3)
print(result)  # 8
Important

Code after return will never execute.

5.2.2 Returning a Single Value

A function may return numbers, strings, lists, booleans, or any Python object.

def greet(name):
    return "Hello, " + name

print(greet("Sukanta"))

5.2.3 Returning Multiple Values (Tuple)

Python allows returning multiple values separated by commas, packed automatically into a tuple.

def operations(a, b):
    return a + b, a - b, a * b

x = operations(10, 5)
print(x)  # (15, 5, 50)

Tuple Unpacking

s, d, m = operations(10, 5)
print(s, d, m)

5.2.4 Void vs Value-Returning Functions

A void function does not return anything. It implicitly returns None.

Void Function Example

def show():
    print("This is a void function.")

print(show())
# Output:
# This is a void function.
# None

Value Returning Example

def square(n):
    return n * n

print(square(6))  # 36

5.2.5 Common Mistakes with Return

Mistake 1 — Using print() Instead of return
def wrong(a, b):
    print(a + b)   # prints, does not return
Mistake 2 — Unreachable Code
def test():
    return 5
    print("Won't execute")

5.2.6 Real-World Examples

Example — Returning a List
def factors(n):
    return [i for i in range(1, n+1) if n % i == 0]

print(factors(12))
Example — Returning a Dictionary
def profile(name, age):
    return {"name": name, "age": age}

print(profile("Sudeshna", 16))
