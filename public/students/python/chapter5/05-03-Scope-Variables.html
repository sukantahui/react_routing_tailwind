5.3 Scope of Variables

Organisation: Coder & AccoTax | Author: Sukanta Hui | Website: www.codernaccotax.co.in

1. What is Variable Scope?

Scope means: Where a variable can be accessed in a program.

Python divides variable scope into four levels (LEGB rule):

  • L – Local: Inside the current function
  • E – Enclosed: Inside nested functions
  • G – Global: Variables defined at the top level
  • B – Built-in: Keywords, built-in functions (e.g., len, sum)

In this chapter, we focus on Local and Global scopes.

2. Local Variables (Highly Detailed)

A local variable is created inside a function and belongs only to that function. It does not exist outside it.

Characteristics:

  • Created when the function starts
  • Destroyed when the function ends
  • Cannot be accessed outside the function
  • Different functions can have variables with the same name without conflict

Basic Example

def message():
    text = "Hello Students"  # local variable
    print(text)

message()
# print(text)  # ❌ ERROR: text is not defined outside the function

Example with Same Local Variable Name in Two Functions

def f1():
    x = 10
    print("f1 x =", x)

def f2():
    x = 20
    print("f2 x =", x)

f1()
f2()

Local Variable is Recreated Every Time

def counter():
    c = 0
    c += 1
    return c

print(counter())
print(counter())
print(counter())
# Output: 1 1 1  (not increasing because c resets each time)

3. Global Variables (Deep Explanation)

A global variable is declared outside any function and can be accessed anywhere.

Properties:

  • Exists for the entire program duration
  • Shared across all functions (read-only unless global keyword used)

Basic Example

x = 50  # global

def show():
    print("Inside function:", x)

show()
print("Outside function:", x)

Using Global Variable in Multiple Functions

count = 0

def inc():
    print("Counter =", count)

def dec():
    print("Counter =", count)

inc()
dec()

4. The global Keyword — Detailed

To modify a global variable inside a function, we must explicitly declare it with the global keyword.

Example: Modifying Global Variable

x = 10

def change():
    global x
    x = 99

change()
print(x)  # 99

Why global is needed?

x = 5

def test():
    x = 100   # This is local, does not affect global

print(x)  # Still 5

Multiple Modification Example

total = 0

def add(n):
    global total
    total += n

add(10)
add(5)
print(total)   # 15

6. Enclosed Scope (Bonus Section)

This scope appears in nested functions. A variable in the outer function is available to the inner function.

Example

def outer():
    x = "Outer Value"
    
    def inner():
        print(x)
    
    inner()

outer()

7. Best Practices (Highly Expanded)

  • Avoid using global variables for data storage. Use return values instead.
  • Prefer local variables — safer and easier to debug.
  • Use global only for constants:
    PI = 3.14, APP_NAME = "Coder & AccoTax"
  • Never create too many global variables.
  • Use function parameters to pass values instead of globals.
  • Use clear naming conventions to avoid conflicts.
  • Nested functions can be used for encapsulation (Enclosed scope).

8. Real-World Example — Banking System

balance = 0

def deposit(amount):
    global balance
    balance += amount
    return balance

def withdraw(amount):
    global balance
    if amount <= balance:
        balance -= amount
        return balance
    return "Insufficient Funds"

print(deposit(1000))
print(withdraw(300))
print(withdraw(800))
