Set A Conceptual & Short Answer Questions

Q1
What is a return value in a function?
Answer

A return value is the data that a function sends back to the caller when it finishes executing.

Explanation

The caller can store this returned value in a variable or use it directly in an expression.

Q2
What is the purpose of the return statement in Python?
Answer

The return statement ends the function and sends a value back to the caller.

Explanation

It is used to give the result of the function’s computation to the part of the program that called it.

Q3
What does a function return if there is no return statement?
Answer

It returns None automatically.

Explanation

In Python, every function returns something; if no explicit value is returned, Python returns the special value None.

Q4
Can a function have more than one return statement? Explain with an example.
Answer

Yes, a function can have multiple return statements, typically inside different branches.

Explanation
def check(n):
    if n > 0:
        return "Positive"
    else:
        return "Non-positive"

Here, one of the return statements will execute depending on the condition.

Q5
What happens to the code written after a return statement inside a function?
Answer

It is never executed (unreachable).

Explanation

Once return runs, the control jumps out of the function immediately.

Q6
Differentiate between printing a value and returning a value from a function.
Answer

Printing only shows the value on the screen; returning sends the value back to be used in code.

Explanation

A returned value can be stored, processed, or passed to other functions, but printed output cannot be reused directly.

Q7
Define a void function with an example.
Answer

A void function does not return any value (implicitly returns None).

Explanation
def show_msg():
    print("Hello")

This function only prints; it does not return a result.

Q8
Define a value-returning function with an example.
Answer

A value-returning function uses return to send a result back.

Explanation
def square(n):
    return n * n

This function returns the square of n, which can be used in expressions.

Q9
Can a function return different data types at different times? Give a simple example.
Answer

Yes, Python functions can return different types on different paths.

Explanation
def demo(x):
    if x >= 0:
        return x
    else:
        return "Negative"

Here, the function may return an int or a str.

Q10
Why is it generally better to use return instead of only print() inside a function?
Answer

Because returned values can be reused, tested, and combined, while printed values are just output.

Explanation

Using return makes functions more flexible and suitable for modular programming.

Q11
What is a tuple? How is it related to returning multiple values?
Answer

A tuple is an immutable ordered collection. When a function returns multiple values, Python packs them in a tuple.

Explanation

Writing return a, b, c actually returns (a, b, c) as a tuple.

Q12
Write the general syntax of a function with a return value.
Answer
def function_name(parameters):
    # statements
    return expression
Explanation

The return line computes and sends the result back to the caller.

Q13
Can a function return another function? Conceptually explain.
Answer

Yes, in Python functions are objects, so one function can return another function.

Explanation

This is used in higher-order functions and closures (e.g. decorators, factory functions).

Q14
What will be the type of the value returned when a function returns a, b, c?
Answer

The function returns a tuple.

Explanation

The values are automatically packed into (a, b, c).

Q15
How can you capture multiple returned values separately in variables? Give an example.
Answer

By using tuple unpacking.

Explanation
def calc(a, b):
    return a + b, a * b

s, p = calc(3, 4)

Here s gets the sum and p gets the product.

Q16
Why is None printed when we do print(my_void_function())?
Answer

Because a void function returns None by default, and print() displays it.

Explanation

The function first prints its own message and then print() prints the value it returned, i.e. None.

Q17
What is meant by "early return" in a function?
Answer

Returning from a function before reaching the last line, often using conditions.

Explanation

It is used to exit quickly when certain conditions are met, instead of nesting many if blocks.

Q18
Can you use return without any value? What does it do?
Answer

Yes. It simply exits the function and returns None.

Explanation

return and return None behave the same.

Q19
How does return help in breaking complex problems into smaller functions?
Answer

Each helper function can return a partial result which another function can reuse.

Explanation

This encourages modular code where every function solves a small part and returns its result to build the full solution.

Q20
What is the difference between return 0 and return None?
Answer

0 is an integer value, whereas None is a special object of type NoneType.

Explanation

They are not the same; comparisons like 0 is None are always False.

    return s
Explanation

We repeatedly take the last digit with % 10 and remove it with integer division.

Q90
Write a function that returns a dictionary with keys "sum", "average", and "count" for a given list of numbers.
Answer
def stats(nums):
    count = len(nums)
    if count == 0:
        return {"sum": 0, "average": 0, "count": 0}
    total = 0
    for n in nums:
        total += n
    avg = total / count
    return {"sum": total, "average": avg, "count": count}
Explanation

We compute these three values once and package them in a dictionary for convenient access.
