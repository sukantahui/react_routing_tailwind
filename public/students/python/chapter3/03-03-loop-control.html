1. break ‚Äî Immediate Exit From Loop

The break statement stops the loop immediately, even if the condition is still true.

Example:

for i in range(1, 10):
    if i == 5:
        break
    print(i)
Output: 1 2 3 4

Flow Diagram

[Loop Start] | [Check Condition] | [Execute Body] | [break?]--Yes--> Exit Loop | No | Next Iteration

2. continue ‚Äî Skip Current Iteration

The continue statement skips the current iteration and moves to the next one.

Example:

for i in range(1, 7):
    if i == 3:
        continue
    print(i)
Output: 1 2 4 5 6

Flow Diagram

[Loop Start] | [Execute Body] | [continue?]--Yes--> Jump to Next Iteration | No | Continue Normally

3. pass ‚Äî Placeholder Statement

pass does nothing. It is used when a statement is required syntactically but we don‚Äôt want any action.

Example:

for i in range(5):
    if i == 2:
        pass  # TODO: implement later
    print(i)

4. Menu-Driven Programs Using Loops

4.1 Simple Calculator Menu

while True:
    print("\nMENU")
    print("1. Add")
    print("2. Subtract")
    print("3. Exit")

    choice = int(input("Enter choice: "))

    if choice == 3:
        print("Exiting...")
        break          # exit menu loop

    a = int(input("Enter first number: "))
    b = int(input("Enter second number: "))

    if choice == 1:
        print("Sum =", a + b)
    elif choice == 2:
        print("Difference =", a - b)
    else:
        print("Invalid choice")

4.2 Menu: List of Tasks

tasks = []

while True:
    print("\n1. Add Task")
    print("2. Show Tasks")
    print("3. Exit")

    ch = int(input("Enter choice: "))

    if ch == 1:
        t = input("Enter task: ")
        tasks.append(t)
    elif ch == 2:
        print("Your Tasks:")
        for x in tasks:
            print("-", x)
    elif ch == 3:
        break
    else:
        print("Invalid option")
