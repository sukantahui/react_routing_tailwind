1. Multiple Choice Questions (MCQs)

  1. Which statement is used for decision-making?
    a) choose   b) if   c) decide   d) switch
  2. Which executes only if condition is False?
    a) if   b) else   c) elif   d) nested if
  3. Which ladder allows multiple checks?
    a) if-only   b) if-else   c) if-elif-else   d) nested-for
  4. Nested if means:
    a) if inside if   b) two ifs separate   c) else inside if   d) both b & c
  5. Ternary expression uses:
    a) ? :   b) if-else in one line   c) eval()   d) when()
  6. Which is valid?
    a) if x > 5 then:   b) if (x>5):   c) if x > 5:   d) if x:>5
  7. Output of: if 0: print("Hi")
    a) Hi   b) 0   c) Nothing   d) Error
  8. Which executes only if all previous conditions are False?
    a) elif   b) else   c) if   d) nested-if
  9. Which is True?
    a) if must have else   b) elif cannot exist alone   c) else can exist alone   d) all
  10. Which is invalid?
    a) if x>0: print(x)   b) if (x>0): print(x)   c) if x>0 print(x)
