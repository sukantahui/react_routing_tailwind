1. What Are Conditional Statements?

Conditional statements allow Python programs to make decisions based on conditions. They control the flow of a program by executing different blocks depending on True/False.

Types of Conditional Statements:
‚Ä¢ if
‚Ä¢ if‚Äìelse
‚Ä¢ if‚Äìelif‚Äìelse

2. The if Statement

Syntax:

if condition:
    statement(s)

Example 1 ‚Äî Check Positive

num = 10 if num > 0: print("Positive number")

Example 2 ‚Äî Check Voting Age

age = 18 if age >= 18: print("Eligible to vote")

3. The if‚Äìelse Statement

Syntax:

if condition:
    true-block
else:
    false-block

Example ‚Äî Even or Odd

n = int(input("Enter number: ")) if n % 2 == 0: print("Even") else: print("Odd")

4. The if‚Äìelif‚Äìelse Ladder

Used when multiple conditions must be checked one after another.

Syntax:

if condition1:
    block1
elif condition2:
    block2
elif condition3:
    block3
else:
    default-block

Example ‚Äî Student Grade System

marks = float(input("Enter %: ")) if marks >= 90: print("A+") elif marks >= 80: print("A") elif marks >= 70: print("B") elif marks >= 60: print("C") elif marks >= 40: print("D") else: print("Fail")

5. Nested if Statements

An if inside another if. Useful for multi-level checks.

Example ‚Äî Admission Eligibility

age = 19 marks = 75 if age >= 18: if marks >= 60: print("Eligible for admission") else: print("Age OK but marks low") else: print("Age less than required")

Example ‚Äî ATM Withdrawal
