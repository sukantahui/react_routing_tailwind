1. Introduction to Loops

Many problems require us to repeat a set of statements multiple times. For example:

  • Print numbers from 1 to 100
  • Calculate the sum of the first N natural numbers
  • Generate a multiplication table
  • Check each element in a list or string
A loop is a control structure that allows us to repeat a block of code as long as a certain condition is true or for each element in a sequence.

Python provides two main looping statements:

  • while loop ‚Äî condition based
  • for loop ‚Äî sequence / range based

2. while Loop

The while loop repeats a block of statements as long as the condition remains True.

2.1 Syntax

while condition:
    statement1
    statement2
    ...

2.2 Flowchart of while loop

+----------------------+ | START | +----------+-----------+ | v +--------+--------+ | Test condition | +--------+--------+ | +------+------+ | | True False | | v v +------+-------+ [Exit loop] | Body of | | loop | +------+-------+ | v [update / change] | +------> back to "Test condition"

2.3 Example ‚Äî Print numbers 1 to 5

i = 1 while i <= 5: print(i) i = i + 1 print("Loop finished.")

Dry Run Table

Iteration i (before check) Condition (i <= 5) Printed i (after update)
1 1 True 1 2
2 2 True 2 3
3 3 True 3 4
4 4 True 4 5
5 5 True 5 6
6 6 False ‚Äî ‚Äî

2.4 Example ‚Äî Sum of first N natural numbers

n = int(input("Enter N: ")) i = 1 s = 0 while i <= n: s = s + i i = i + 1 print("Sum =", s)

2.5 Infinite Loop (and how to avoid)

# Infinite loop example (do NOT run as it is) i = 1 while i <= 5: print(i) # forgot to update i
Important: Always ensure the loop condition will eventually become False. Otherwise, the loop may run forever (infinite loop).

3. for Loop and range()

Python‚Äôs for loop is used to iterate over a sequence (list, tuple, string, range, etc.).

3.1 Syntax with range()

for variable in range(start, stop, step):
    statements

‚Ä¢ start: starting value (default 0)
‚Ä¢ stop: loop stops before this value
‚Ä¢ step: increment (default 1)

3.2 Example ‚Äî Print 0 to 4

for i in range(5): # start = 0, stop = 5, step = 1 print(i)

3.3 Example ‚Äî Print 1 to 5

for i in range(1, 6): print(i)

3.4 Example ‚Äî Print even numbers from 2 to 10

for i in range(2, 11, 2): print(i, end=" ")

3.5 Using for loop with a list

fruits = ["apple", "banana", "mango"] for item in fruits: print("I like", item)

3.6 Using for loop with a string

word = "PYTHON" for ch in word: print(ch)

4. Loop Control Statements: break, continue, pass

4.1 break

break immediately terminates the nearest enclosing loop.

Example ‚Äî Stop when i becomes 5
for i in range(1, 11): if i == 5: break print(i) print("Loop exited using break.")

4.2 continue

continue skips the remaining statements in current iteration and jumps to the next iteration.

Example ‚Äî Skip 5
for i in range(1, 8): if i == 5: continue print(i)

4.3 pass

pass does nothing ‚Äî it is a null statement used as a placeholder.

for i in range(5): if i == 3: pass # to be implemented later print(i)
Use pass when the statement is required syntactically, but you don't want any action yet.

5. Nested Loops

A nested loop is a loop inside another loop. Commonly used for:

  • Pattern printing (stars, numbers)
  • Working with 2D data (matrices)
  • Comparing each pair of values

5.1 Syntax

for i in range(...):
    for j in range(...):
        statements

5.2 Example ‚Äî Simple nested loop

for i in range(1, 4): for j in range(1, 4): print("(", i, ",", j, ")", end=" ") print()

6. Pattern Printing Using Nested Loops

6.1 Square of Stars

Output
***** ***** ***** ***** *****
Code
n = 5 for i in range(n): for j in range(n): print("*", end="") print()

6.2 Right-Angled Triangle (Stars)

Output
* ** *** **** *****
Code
n = 5 for i in range(1, n + 1): for j in range(i): print("*", end="") print()

6.3 Number Triangle

Output
1 12 123 1234 1235
Code
n = 5 for i in range(1, n + 1): for j in range(1, i + 1): print(j, end="") print()

6.4 Inverted Triangle

Output
***** **** *** ** *
Code
n = 5 for i in range(n, 0, -1): for j in range(i): print("*", end="") print()

6.5 Simple Pyramid

Output
* *** ***** ******* *********
Code
n = 5 for i in range(1, n + 1): # print spaces for s in range(n - i): print(" ", end="") # print stars for k in range(2 * i - 1): print("*", end="") print()

7. Classic Programs with Loops

7.1 Factorial of a Number

n = int(input("Enter a number: ")) fact = 1 for i in range(1, n + 1): fact = fact * i print("Factorial of", n, "=", fact)

7.2 Fibonacci Series

n = int(input("How many terms? ")) a, b = 0, 1 count = 0 while count < n: print(a, end=" ") a, b = b, a + b count += 1

7.3 Prime Number Check

n = int(input("Enter a number: ")) if n <= 1: print("Not prime") else: is_prime = True for i in range(2, n): if n % i == 0: is_prime = False break if is_prime: print("Prime") else: print("Not prime")

7.4 Reverse Digits of a Number

n = int(input("Enter a number: ")) rev = 0 while n > 0: digit = n % 10 rev = rev * 10 + digit n = n // 10 print("Reversed number:", rev)

7.5 Multiplication Table

num = int(input("Enter a number: ")) for i in range(1, 11): print(num, "x", i, "=", num * i)

11. Summary ‚Äî Unit 3.2 Looping Statements

  • Loops allow repeated execution of a block of code.
  • while loop repeats while a condition remains True.
  • for loop iterates over a sequence like range(), list or string.
  • break exits the loop, continue skips the current iteration, and pass does nothing.
  • Nested loops are powerful for pattern printing and 2D data processing.
  • Most fundamental programming tasks (factorial, Fibonacci, prime check, tables) can be done using loops.
