1. Multiple Choice Questions (MCQs)

  1. Which loop in Python is condition-controlled?
    a) for   b) while   c) do-while   d) None
  2. What is the output of range(1, 5)?
    a) 1 2 3 4 5   b) 1 2 3 4   c) 0 1 2 3 4   d) 2 3 4 5
  3. Which statement stops the loop immediately?
    a) exit   b) break   c) continue   d) stop
  4. Which statement skips the current iteration?
    a) skip   b) break   c) continue   d) next
  5. Which statement does nothing?
    a) pass   b) break   c) continue   d) null
  6. How many times will this loop run?
    for i in range(2, 10, 2):
    print(i)
    a) 3   b) 4   c) 5   d) 6
  7. Which loop is best for fixed number of iterations?
    a) while   b) for   c) do-while
  8. What is the last value printed?
    for i in range(3):
    print(i)
    a) 0   b) 1   c) 2   d) 3
  9. Which loop can generate an infinite loop easily?
    a) for   b) while
  10. What is the output?
    for i in range(5):
    if i == 3:
        break
    print(i)
    a) 0 1 2 3
    b) 0 1 2
    c) 3
    d) none

6. Loop Flow Diagrams (ASCII)

6.1 while loop flow

+-----------+ | START | +-----+-----+ | v [Check cond] | +--+--+ | | True False | | v v [Body] END | v [Update] | +---> back to condition

6.2 for loop flow

START | v [Take first value from range/list] | v [Loop Body] | v [Next Value Exists?] | | Yes No | | v v Repeat END

6.3 break working

[Loop] | [Condition True] | [break?]--Yes-->Exit Loop | No | Continue Loop
