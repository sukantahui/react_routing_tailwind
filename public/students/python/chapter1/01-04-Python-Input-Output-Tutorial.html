1.4 Input & Output Basics

Learn how to take user input using input(), display output using print(), and combine both to build simple interactive programs.

1.4.1 Taking Input Using input()

The input() function allows users to type values during program execution. Whatever the user enters is always taken as a string.

# Taking simple input
name = input("Enter your name: ")
print("Hello", name)
Important

input() returns data as string only. If you want numeric input, you must convert it manually using int() or float().

# Taking numeric input
age = int(input("Enter your age: "))
print("Next year you will be", age + 1)

1.4.2 Output Using print()

The print() function is used to display messages, variables, and results.

Basic Printing

print("Welcome to Python")
print(25)
print(3.14)

Printing Multiple Values

x = 10
y = 20
print("The values are", x, "and", y)

Formatted Output – f‑strings

name = "Riya"
age = 20
print(f"My name is {name} and I am {age} years old.")

Using sep and end parameters

# Custom separator
print(1, 2, 3, sep=" - ")  # 1 - 2 - 3

# Custom ending
print("Hello", end="...")
print("World")

1.4.3 Combining Input, Processing & Output

Here are simple programs that take user input, perform calculations, and show results.

Example 1: Add Two Numbers

a = int(input("Enter first number: "))
b = int(input("Enter second number: "))

result = a + b
print("The sum is:", result)

Example 2: Area of a Circle

r = float(input("Enter radius: "))
area = 3.1416 * r * r
print(f"Area = {area}")

Example 3: Greeting Message

name = input("Enter your name: ")
time = input("Morning / Afternoon / Evening? ")

print(f"Good {time}, {name}! Welcome to Python learning.")
