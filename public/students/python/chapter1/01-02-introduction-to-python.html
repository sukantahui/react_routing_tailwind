1.2 Introduction to Python

What is Python?

Python is a high-level, general-purpose, interpreted programming language created by Guido van Rossum in 1991. It is one of the most popular programming languages in the world because of its simplicity, readability, and huge community support.

Key idea: Python looks like English and is easy to understand even for absolute beginners.

Features of Python

  • Simple & readable syntax (looks like English)
  • Interpreted language (no need to compile)
  • Platform-independent (runs on Windows, Linux, Mac)
  • Object-oriented yet also supports structured programming
  • Large standard library
  • Huge community + easy availability of packages
  • Used in emerging technologies: AI, ML, Data Science, Web Dev, Automation

Applications of Python

  • Web Development (Django, Flask)
  • Machine Learning & AI (TensorFlow, Scikit-learn)
  • Data Science & Data Analysis (Pandas, NumPy)
  • Automation & Scripting
  • Game Development
  • Cybersecurity & Networking
  • Desktop Apps
  • IOT & Robotics
Example: A simple Python program print("Welcome to Python Programming!")

Python Interactive Mode vs Script Mode

1. Interactive Mode

Useful for quick testing. Shows output immediately.

Example (Interactive Mode): >>> print("Hello") Hello >>> 5 + 10 15

2. Script Mode

Used for writing full programs in a file.

Example (Script Mode): # filename: myprogram.py a = 10 b = 20 print("Sum =", a + b)
Important: Save your file with the extension .py.

Understanding Indentation

Python uses indentation (spaces / tabs) to define blocks of code. Indentation is not optional — it is a part of Python’s syntax.

Correct indentation: if 10 > 5: print("True") Incorrect indentation: if 10 > 5: print("True") # Error!
Use **4 spaces** indentation for clean, error-free code.

Variables & Identifiers

A variable is a name that stores a value in memory. An identifier is the name given to a variable, function, class, etc.

Rules for Identifiers

  • Must start with a letter or underscore (_)
  • Cannot start with a number
  • No special characters allowed: @, #, %, etc.
  • Case-sensitive (Age ≠ age)
  • Cannot use Python keywords
Examples: name = "Sukanta" age = 25 _total = 450 RateOfInterest = 8.5

Keywords & Reserved Words

Keywords are special reserved words that have predefined meaning in Python. They cannot be used as variable names.

Some Python Keywords: False, True, None, if, else, elif, while, for, break, continue, import, from, class, def, return, try, except, finally, global

Comments in Python

Comments are notes in the program that Python ignores. They help explain the code.

Single-line Comment

# This is a comment print("Hello Students")

Multi-line Comment

""" This program adds two numbers. Author: Sukanta Hui """ a = 10 b = 20 print(a + b)
Comments improve readability and help during debugging.
